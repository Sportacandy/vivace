# Napisy

Menu **Napisy** zarządza napisami.

- **Ścieżka** — wybierz osadzoną ścieżkę napisów lub **Wył.**, aby je ukryć.
- **Wczytaj napisy…** — wczytaj zewnętrzny plik `.srt`, `.vtt` lub `.ass`/`.ssa`.
  Vivace może też automatycznie wczytać napisy leżące obok wideo
  (*Ustawienia ▸ Napisy*).
- **Znajdź napisy w OpenSubtitles…** — wyszukaj i pobierz napisy online. Wymaga
  klucza API OpenSubtitles, wprowadzanego w *Ustawienia ▸ Sieć*.
- **Usuń napisy** — usuń obecnie wczytane napisy zewnętrzne.
- **Opóźnienie −** / **Opóźnienie +** — przesuń synchronizację napisów wcześniej lub później.
- **Ustaw opóźnienie…** — wprowadź dokładne opóźnienie napisów.

Czcionkę, rozmiar i położenie napisów ustawia się w *Ustawienia ▸ Napisy*.

**Uwaga:** napisy wczytane zewnętrznie obsługuje się przez *Wczytaj* / *Usuń* —
nie są wymienione w *Ścieżka* (która pokazuje ścieżki osadzone i *Wył.*).
