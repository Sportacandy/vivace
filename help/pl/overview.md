# Pierwsze kroki

Witaj w **Vivace** (wymawiane *wi-WA-cze*), szybkim odtwarzaczu multimedialnym w
czystym Qt. To jest wbudowana pomoc offline Vivace. Wybierz temat z listy po
lewej lub zacznij od podstaw poniżej.

## Odtwórz coś

- **Plik** — *Otwórz ▸ Plik…* (`Ctrl+O`), albo przeciągnij plik na okno.
- **Folder** — *Otwórz ▸ Katalog…* odtwarza wszystko w folderze.
- **Adres URL / strumień** — *Otwórz ▸ URL…* (`Ctrl+U`).
- **DVD** — *Otwórz ▸ Płyta ▸ DVD…*, albo upuść folder zawierający `VIDEO_TS`.

Upuszczenie kilku plików naraz wczytuje je jako listę odtwarzania.

## Podstawowe skróty

| Działanie | Skrót |
| --- | --- |
| Odtwarzanie / pauza | `Spacja` |
| Do tyłu / do przodu | `←` / `→` |
| Ciszej / głośniej | `↓` / `↑` |
| Wycisz | `M` |
| Pełny ekran | `F` (lub dwukrotne kliknięcie wideo) |
| Wyjdź z pełnego ekranu | `Esc` |
| Zrób zrzut ekranu | `S` |
| Lista odtwarzania | `F9` |
| Ustawienia | `Ctrl+P` |

Każdy skrót można zmienić w *Ustawienia ▸ Klawiatura i mysz*.

## Pomoc według menu

Każde menu Vivace ma własną stronę:

- [Otwórz](menu-open.md) — otwieranie plików, folderów, płyt, adresów URL, ulubionych
- [Odtwarzanie](menu-play.md) — odtwarzanie, przewijanie, prędkość, powtarzanie A-B
- [Wideo](menu-video.md) — pełny ekran, rozmiar, powiększenie, proporcje, korektor, zrzuty
- [Dźwięk](menu-audio.md) — ścieżki, głośność, opóźnienie
- [Napisy](menu-subtitles.md) — ścieżki, pliki zewnętrzne, synchronizacja
- [Przeglądaj](menu-browse.md) — tytuły DVD, rozdziały, zakładki
- [Widok](menu-view.md) — informacje o pliku, lista odtwarzania, OSD
- [Opcje](menu-options.md) — ustawienia, ikona w zasobniku, paski narzędzi, pasek stanu
- [Pomoc](menu-help.md) — ta pomoc, aktualizacje, informacje
