# Àudio

El menú **Àudio** controla el so.

- **Pista** — triar entre les pistes d'àudio incrustades del fitxer. Es pot
  seleccionar automàticament un idioma preferit a *Preferències ▸ General*.
- **Silencia** (`M`) — activar/desactivar el so.
- **Volum −** / **Volum +** (`↓` / `↑`) — canviar el volum; la mida del pas es
  configura a *Preferències ▸ General*.
- **Retard −** / **Retard +** — avançar o endarrerir l'àudio per corregir la sincronia labial.
- **Estableix el retard…** — introduir un retard d'àudio exacte per fitxer.

## Sobre el retard d'àudio

El retard aplicat a un fitxer és la suma d'un retard **per dispositiu** (recordat
per a cada dispositiu de sortida d'àudio, útil per a la latència del Bluetooth,
configurat a *Preferències ▸ General ▸ Àudio*) i el retard **per fitxer** d'aquest
menú. El Vivace aplica el desfasament endarrerint el vídeo perquè coincideixi, de
manera que els retards negatius (l'àudio ha d'arribar abans) funcionen com s'espera.
