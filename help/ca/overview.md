# Primers passos

Us donem la benvinguda al **Vivace** (pronunciat *vi-VA-txe*), un reproductor
multimèdia ràpid i en Qt pur. Aquesta és l'ajuda fora de línia integrada del
Vivace. Trieu un tema de la llista de l'esquerra o comenceu amb els conceptes
bàsics a continuació.

## Reproduir alguna cosa

- **Un fitxer** — *Obre ▸ Fitxer…* (`Ctrl+O`), o arrossegueu un fitxer a la finestra.
- **Una carpeta** — *Obre ▸ Directori…* reprodueix tot el que hi ha en una carpeta.
- **Un URL / flux** — *Obre ▸ URL…* (`Ctrl+U`).
- **Un DVD** — *Obre ▸ Disc ▸ DVD…*, o deixeu-hi anar una carpeta que contingui `VIDEO_TS`.

Deixar anar diversos fitxers alhora els carrega com una llista de reproducció.

## Dreceres essencials

| Acció | Drecera |
| --- | --- |
| Reprodueix / pausa | `Espai` |
| Enrere / endavant | `←` / `→` |
| Abaixa / apuja el volum | `↓` / `↑` |
| Silencia | `M` |
| Pantalla completa | `F` (o feu doble clic al vídeo) |
| Surt de la pantalla completa | `Esc` |
| Fes una captura | `S` |
| Llista de reproducció | `F9` |
| Preferències | `Ctrl+P` |

Cada drecera es pot canviar a *Preferències ▸ Teclat i ratolí*.
