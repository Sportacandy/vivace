# Първи стъпки

Добре дошли във **Vivace** (произнася се *ви-ВА-че*) — бърз медиен плейър изцяло
на Qt. Това е вградената офлайн помощ на Vivace. Изберете тема от списъка вляво
или започнете с основите по-долу.

## Възпроизвеждане на нещо

- **Файл** — *Отваряне ▸ Файл…* (`Ctrl+O`), или плъзнете файл върху прозореца.
- **Папка** — *Отваряне ▸ Директория…* възпроизвежда всичко в дадена папка.
- **URL / поток** — *Отваряне ▸ URL…* (`Ctrl+U`).
- **DVD** — *Отваряне ▸ Диск ▸ DVD…*, или пуснете папка, съдържаща `VIDEO_TS`.

Пускането на няколко файла наведнъж ги зарежда като списък за изпълнение.

## Основни клавишни комбинации

| Действие | Комбинация |
| --- | --- |
| Възпроизвеждане / пауза | `Интервал` |
| Назад / напред | `←` / `→` |
| По-тихо / по-силно | `↓` / `↑` |
| Заглушаване | `M` |
| Цял екран | `F` (или двойно щракване върху видеото) |
| Изход от цял екран | `Esc` |
| Заснемане на екрана | `S` |
| Списък за изпълнение | `F9` |
| Настройки | `Ctrl+P` |

Всяка комбинация може да се промени в *Настройки ▸ Клавиатура и мишка*.
