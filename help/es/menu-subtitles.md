# Subtítulos

El menú **Subtítulos** gestiona los subtítulos.

- **Pista** — elegir una pista de subtítulos integrada, o **Desactivado** para ocultarlos.
- **Cargar subtítulos…** — cargar un archivo externo `.srt`, `.vtt` o
  `.ass`/`.ssa`. Vivace también puede cargar automáticamente un subtítulo situado
  junto al vídeo (*Preferencias ▸ Subtítulos*).
- **Buscar subtítulos en OpenSubtitles…** — buscar y descargar subtítulos en
  línea. Requiere una clave de API de OpenSubtitles, introducida en
  *Preferencias ▸ Red*.
- **Descargar subtítulos** — quitar el subtítulo externo cargado actualmente.
- **Retardo −** / **Retardo +** — adelantar o retrasar la sincronización de los subtítulos.
- **Establecer retardo…** — introducir un retardo de subtítulos exacto.

La fuente, el tamaño y la posición de los subtítulos se configuran en
*Preferencias ▸ Subtítulos*.

**Nota:** un subtítulo cargado externamente se controla con *Cargar* /
*Descargar* — no aparece en *Pista* (que muestra las pistas integradas y *Desactivado*).
