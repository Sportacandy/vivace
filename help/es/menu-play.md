# Reproducir

El menú **Reproducir** controla la reproducción y la búsqueda.

## Transporte

- **Reproducir**, **Pausar**, **Detener** — `Espacio` alterna reproducir/pausar;
  `Ctrl+Espacio` detiene.
- **Avanzar fotograma** (`.`) y **Retroceder fotograma** — avanzar o retroceder un fotograma.
- **Anterior** / **Siguiente** — moverse entre elementos de la lista de reproducción.

## Búsqueda

Se ofrecen tres tamaños de salto en ambos sentidos (**Retroceder** /
**Avanzar**), mostrados con su duración actual. Ajuste los tamaños en
*Preferencias ▸ Interfaz ▸ Búsqueda*. También puede buscar con `←` / `→`, la
barra de búsqueda o la rueda del ratón sobre el vídeo.

## Velocidad

- **Velocidad normal** (`Retroceso`), **Mitad de velocidad**, **Doble velocidad**.
- **Velocidad −10 % / +10 %** y pasos más finos de ±4 % / ±1 %.
- **Compensación de tono** — mantiene las voces naturales a velocidades no normales.

La velocidad de reproducción es por archivo y vuelve a la normal con cada archivo nuevo.

## Sección A-B

Repetir parte de un archivo:

- **Poner marca A**, **Poner marca B** — marcar el inicio y el fin.
- **Repetir** — repetir la región A-B (la sección solo se repite mientras esto esté activado).
- **Borrar marcas A-B** — eliminarlas. Las marcas se reinician con cada archivo nuevo.

**Ir a…** (ir a un momento concreto) está previsto para una versión posterior.

## Transmisión

**Transmitir a ▸ Teléfono/tableta…** inicia un pequeño servidor web integrado
para que un teléfono o tableta en la misma red pueda abrir una página y
reproducir el archivo que Vivace está reproduciendo — escaneando el código QR
o escribiendo la dirección mostrada. El puerto está fijado en *Preferencias ▸
Red ▸ Transmisión* (no editable en cada sesión), para que pueda permitirlo una
sola vez en su cortafuegos/router. Solo se transmite el archivo que se está
reproduciendo, y solo mientras la transmisión esté activa.
