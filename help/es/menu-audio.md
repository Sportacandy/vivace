# Audio

El menú **Audio** controla el sonido.

- **Pista** — elegir entre las pistas de audio integradas del archivo. Se puede
  autoseleccionar un idioma preferido en *Preferencias ▸ General*.
- **Silenciar** (`M`) — activar/desactivar el sonido.
- **Volumen −** / **Volumen +** (`↓` / `↑`) — cambiar el volumen; el tamaño del
  paso se configura en *Preferencias ▸ General*.
- **Retardo −** / **Retardo +** — adelantar o retrasar el audio para corregir la sincronía labial.
- **Establecer retardo…** — introducir un retardo de audio exacto por archivo.

## Acerca del retardo de audio

El retardo aplicado a un archivo es la suma de un retardo **por dispositivo**
(recordado para cada dispositivo de salida de audio, útil para la latencia
Bluetooth, configurado en *Preferencias ▸ General ▸ Audio*) y el retardo **por
archivo** de este menú. Vivace aplica el desfase retrasando el vídeo para que
coincida, de modo que los retardos negativos (el audio debe llegar antes)
funcionan como se espera.
