# Primeros pasos

Le damos la bienvenida a **Vivace** (pronunciado *vi-VA-che*), un reproductor
multimedia rápido y en Qt puro. Esta es la ayuda sin conexión integrada de
Vivace. Elija un tema de la lista de la izquierda o comience con lo básico a
continuación.

## Reproducir algo

- **Un archivo** — *Abrir ▸ Archivo…* (`Ctrl+O`), o arrastre un archivo a la ventana.
- **Una carpeta** — *Abrir ▸ Directorio…* reproduce todo lo de una carpeta.
- **Una URL / emisión** — *Abrir ▸ URL…* (`Ctrl+U`).
- **Un DVD** — *Abrir ▸ Disco ▸ DVD…*, o suelte una carpeta que contenga `VIDEO_TS`.

Soltar varios archivos a la vez los carga como una lista de reproducción.

## Atajos esenciales

| Acción | Atajo |
| --- | --- |
| Reproducir / pausar | `Espacio` |
| Retroceder / avanzar | `←` / `→` |
| Bajar / subir volumen | `↓` / `↑` |
| Silenciar | `M` |
| Pantalla completa | `F` (o doble clic en el vídeo) |
| Salir de pantalla completa | `Esc` |
| Hacer una captura | `S` |
| Lista de reproducción | `F9` |
| Preferencias | `Ctrl+P` |

Cada atajo puede cambiarse en *Preferencias ▸ Teclado y ratón*.
