# Početak rada

Dobro došli u **Vivace** (izgovara se *vi-VA-če*), brzi medijski reproduktor u
čistom Qt-u. Ovo je ugrađena izvanmrežna pomoć Vivacea. Odaberite temu s popisa
slijeva ili počnite s osnovama u nastavku.

## Reprodukcija nečega

- **Datoteka** — *Otvori ▸ Datoteka…* (`Ctrl+O`), ili povucite datoteku na prozor.
- **Mapa** — *Otvori ▸ Direktorij…* reproducira sve u mapi.
- **URL / stream** — *Otvori ▸ URL…* (`Ctrl+U`).
- **DVD** — *Otvori ▸ Disk ▸ DVD…*, ili ispustite mapu koja sadrži `VIDEO_TS`.

Ispuštanje više datoteka odjednom učitava ih kao popis za reprodukciju.

## Osnovni prečaci

| Radnja | Prečac |
| --- | --- |
| Reprodukcija / pauza | `Razmaknica` |
| Natrag / naprijed | `←` / `→` |
| Tiše / glasnije | `↓` / `↑` |
| Utišaj | `M` |
| Cijeli zaslon | `F` (ili dvoklik na videozapis) |
| Izlaz iz cijelog zaslona | `Esc` |
| Snimi zaslon | `S` |
| Popis za reprodukciju | `F9` |
| Postavke | `Ctrl+P` |

Svaki se prečac može promijeniti u *Postavke ▸ Tipkovnica i miš*.

## Pomoć po izborniku

Svaki izbornik Vivacea ima vlastitu stranicu:

- [Otvori](menu-open.md) — otvaranje datoteka, mapa, diskova, URL-ova, favorita

- [Videozapis](menu-video.md) — cijeli zaslon, veličina, zumiranje, omjer slike, ekvilizator, snimke
- [Zvuk](menu-audio.md) — zapisi, glasnoća, kašnjenje
- [Titlovi](menu-subtitles.md) — zapisi, vanjske datoteke, usklađivanje
- [Pregledaj](menu-browse.md) — DVD naslovi, poglavlja, oznake
- [Prikaz](menu-view.md) — informacije o mediju, popis za reprodukciju, OSD
- [Opcije](menu-options.md) — postavke, ikona u traci sustava, alatne trake, statusna traka
- [Pomoć](menu-help.md) — ova pomoć, ažuriranja, o programu
