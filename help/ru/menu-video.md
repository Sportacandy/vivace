# Видео

Меню **Видео** настраивает отображение изображения. Эти настройки применяются к
текущему файлу и сбрасываются для каждого нового.

- **Дорожка** — выбрать одну из встроенных видеодорожек файла.
- **Полный экран** (`F`) — переключить полный экран; `Esc` выходит из него. Можно
  также дважды щёлкнуть по видео.
- **Размер** — изменить размер окна до **50 % / 100 % / 200 %** от собственного
  разрешения видео.
- **Масштаб и панорама**
  - **Сброс** (`Shift+E`), **Уменьшить** (`W`), **Увеличить** (`E`).
  - **Сдвинуть влево / вправо / вверх / вниз** (`Alt`+стрелки), чтобы сдвинуть
    увеличенное изображение.
- **Соотношение сторон** — принудительно задать соотношение (Авто, 4:3, 16:9, …).
- **Поворот** — Нет, 90° по часовой, 90° против часовой или 180°.
- **Отразить по вертикали** / **Отразить по горизонтали** — перевернуть изображение.
- **Эквалайзер…** (`Ctrl+E`) — яркость, контраст, оттенок, насыщенность и гамма.
  *Установить как значения по умолчанию* применяет ваши настройки к будущим файлам.
- **Снимок экрана** (`S`) — сохранить текущий кадр. Папка и формат изображения
  задаются в *Настройки ▸ Общие*.
