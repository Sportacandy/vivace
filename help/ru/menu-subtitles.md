# Субтитры

Меню **Субтитры** управляет субтитрами.

- **Дорожка** — выбрать встроенную дорожку субтитров или **Выкл.**, чтобы их скрыть.
- **Загрузить субтитры…** — загрузить внешний файл `.srt`, `.vtt` или
  `.ass`/`.ssa`. Vivace также может автоматически загрузить субтитры рядом с
  видео (*Настройки ▸ Субтитры*).
- **Найти субтитры на OpenSubtitles…** — искать и скачивать субтитры в сети.
  Для этого нужен ключ API OpenSubtitles, вводимый в *Настройки ▸ Сеть*.
- **Выгрузить субтитры** — убрать текущие загруженные внешние субтитры.
- **Задержка −** / **Задержка +** — сдвинуть тайминг субтитров раньше или позже.
- **Задать задержку…** — ввести точную задержку субтитров.

Шрифт, размер и положение субтитров задаются в *Настройки ▸ Субтитры*.

**Примечание:** внешне загруженные субтитры управляются через *Загрузить* /
*Выгрузить* — они не отображаются в *Дорожка* (там показаны встроенные дорожки и *Выкл.*).
