# Звук

Меню **Звук** управляет звуком.

- **Дорожка** — выбрать одну из встроенных звуковых дорожек файла. Предпочитаемый
  язык можно выбрать автоматически в *Настройки ▸ Общие*.
- **Без звука** (`M`) — включить/выключить звук.
- **Тише** / **Громче** (`↓` / `↑`) — изменить громкость; размер шага задаётся в
  *Настройки ▸ Общие*.
- **Задержка −** / **Задержка +** — сдвинуть звук раньше или позже, чтобы поправить синхронизацию губ.
- **Задать задержку…** — ввести точную задержку звука для файла.

## О задержке звука

Задержка, применяемая к файлу, — это сумма задержки **для устройства**
(запоминается для каждого устройства вывода звука; полезно при задержке
Bluetooth, задаётся в *Настройки ▸ Общие ▸ Аудио*) и задержки **для файла** из
этого меню. Vivace применяет смещение, задерживая видео для совпадения, поэтому
отрицательные задержки (звук должен идти раньше) работают как ожидается.
