# Справка

Меню **Справка**.

- **Содержание** (`F1`) — открыть это окно справки. Список слева — оглавление;
  щёлкните тему, чтобы прочитать её справа. Ссылки внутри страницы открывают
  связанную тему здесь; внешние ссылки открываются в веб-браузере.
- **Проверить обновления** — узнать, доступна ли более новая версия Vivace.
  Автоматические проверки включаются в *Настройки ▸ Обновления*.
- **Установить / обновить поддержку YouTube** — скачать или обновить `yt-dlp`,
  вспомогательную программу, которую Vivace использует для воспроизведения
  ссылок YouTube (см. *Настройки ▸ Сеть*).
- **О программе Vivace** — версия, версии Qt, благодарности и лицензия.

Vivace — созданный с нуля медиаплеер полностью на Qt, вдохновлённый SMPlayer,
выпущенный под лицензией **GPL-3.0-or-later**.
