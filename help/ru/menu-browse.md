# Обзор

Меню **Обзор** перемещается внутри текущего медиа — в основном DVD и главы — и
управляет закладками.

- **Меню DVD** — вернуться к меню диска (доступно, пока воспроизводится DVD с
  меню). В меню щёлкните кнопку или используйте стрелки и `Enter`. Меню DVD
  экспериментальны; поведение настраивается в *Настройки ▸ Приводы*.
- **Заголовок** — для DVD перейти к конкретному заголовку (с его длительностью).
- **Главы** — перейти к главе. Для файлов главы читаются из контейнера (mkv/mp4);
  для DVD — с диска. Метки глав также отображаются засечками на ползунке перемотки.
- **Закладки** — ваши именованные позиции в текущем файле:
  - **Добавить закладку** (`Ctrl+A`) — сделать закладку на текущем времени.
  - **Изменить закладки…** — переименовать или удалить их; щёлкните по одной, чтобы перейти туда.

  Закладки хранятся для каждого файла (для DVD — для каждого заголовка) отдельно
  от истории возобновления.
