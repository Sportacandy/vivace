# Параметры командной строки

Vivace можно запустить из терминала с медиафайлом и несколькими параметрами.
Используйте длинные параметры в стиле GNU (короткие псевдонимы указаны в скобках).

```
vivace [параметры] [медиа]
```

- **медиа** — путь к файлу или URL для воспроизведения. Здесь работает всё, что
  Vivace обычно может открыть (локальный файл, папка, `VIDEO_TS`, сетевой поток, URL YouTube).

## Параметры

- **`--help` (`-h`)** — показать список параметров и выйти.
- **`--version` (`-v`)** — показать версию и выйти.
- **`--fullscreen` (`-f`)** — запустить в полноэкранном режиме.
- **`--ontop` (`-t`)** — держать окно поверх других окон.
- **`--close-at-end` (`-c`)** — закрыть Vivace по окончании воспроизведения.
- **`--sub <файл>` (`-s`)** — загрузить указанный файл субтитров.
- **`--start <время>` (`-b`)** — начать воспроизведение с `<время>`, заданного
  как `h:m:s`, `m:s` или число секунд.
- **`--pos <x,y>`** — разместить окно в позиции экрана `x,y`.
- **`--size <ш,в>`** — задать размер окна `ш`×`в`.

## Примеры

```
vivace movie.mkv
vivace --fullscreen --start 1:30 movie.mkv
vivace -f -b 90 -s movie.en.srt movie.mkv
vivace --pos 100,100 --size 1280,720 movie.mp4
```

**Примечание (Windows):** Vivace — приложение с графическим интерфейсом, поэтому
`--help` / `--version` выводятся в терминал, из которого вы его запустили
(приглашение оболочки может вернуться первым, так что текст может появиться
после). Когда включён режим одного экземпляра (*Настройки ▸ Интерфейс ▸
Экземпляры*), повторный запуск с медиафайлом и этими параметрами применяет их к
уже работающему окну.
