# Начало работы

Добро пожаловать в **Vivace** (произносится *ви-ВА-че*) — быстрый медиаплеер
полностью на Qt. Это встроенная автономная справка Vivace. Выберите тему из
списка слева или начните с основ ниже.

## Что-нибудь воспроизвести

- **Файл** — *Открыть ▸ Файл…* (`Ctrl+O`) или перетащите файл в окно.
- **Папку** — *Открыть ▸ Каталог…* воспроизводит всё содержимое папки.
- **URL / поток** — *Открыть ▸ URL…* (`Ctrl+U`).
- **DVD** — *Открыть ▸ Диск ▸ DVD…* или перетащите папку с `VIDEO_TS`.

Перетаскивание нескольких файлов сразу загружает их как плейлист.

## Основные сочетания клавиш

| Действие | Сочетание |
| --- | --- |
| Воспроизведение / пауза | `Пробел` |
| Назад / вперёд | `←` / `→` |
| Тише / громче | `↓` / `↑` |
| Без звука | `M` |
| Полный экран | `F` (или двойной щелчок по видео) |
| Выход из полного экрана | `Esc` |
| Сделать снимок экрана | `S` |
| Плейлист | `F9` |
| Настройки | `Ctrl+P` |

Любое сочетание можно изменить в *Настройки ▸ Клавиатура и мышь*.

## Справка по меню

У каждого меню Vivace есть своя страница:

- [Открыть](menu-open.md) — открытие файлов, папок, дисков, URL, избранного
- [Воспроизведение](menu-play.md) — воспроизведение, перемотка, скорость, повтор A-B
- [Видео](menu-video.md) — полный экран, размер, масштаб, соотношение сторон, эквалайзер, снимки
- [Звук](menu-audio.md) — дорожки, громкость, задержка
- [Субтитры](menu-subtitles.md) — дорожки, внешние файлы, синхронизация
- [Обзор](menu-browse.md) — заголовки DVD, главы, закладки
- [Вид](menu-view.md) — сведения о медиа, плейлист, экранное меню
- [Параметры](menu-options.md) — настройки, значок в трее, панели инструментов, строка состояния
- [Справка](menu-help.md) — эта справка, обновления, о программе
