# Subtitles

The **Subtitles** menu manages captions.

- **Track** — choose an embedded subtitle track, or **Off** to hide subtitles.
- **Load subtitles…** — load an external `.srt`, `.vtt` or `.ass`/`.ssa` file.
  Vivace can also auto-load a subtitle sitting next to the video
  (*Preferences ▸ Subtitles*).
- **Find subtitles at OpenSubtitles…** — search and download subtitles online.
  This needs an OpenSubtitles API key, entered in *Preferences ▸ Network*.
- **Unload subtitles** — remove the currently loaded external subtitle.
- **Delay −** / **Delay +** — shift subtitle timing earlier or later.
- **Set delay…** — enter an exact subtitle delay.

Subtitle font, size and position are set in *Preferences ▸ Subtitles*.

**Note:** an externally loaded subtitle is controlled with *Load* / *Unload* —
it is not listed under *Track* (which shows the embedded tracks and *Off*).
