# Help

The **Help** menu.

- **Contents** (`F1`) — open this help window. The list on the left is the
  table of contents; click a topic to read it on the right. Links within a page
  open the related topic here; external links open in your web browser.
- **Check for updates** — see whether a newer Vivace is available. Automatic
  checks can be enabled in *Preferences ▸ Updates*.
- **Install / Update YouTube support** — download or update `yt-dlp`, the helper
  Vivace uses to play YouTube links (see *Preferences ▸ Network*).
- **About Vivace** — version, Qt versions, credits and license.

Vivace is a ground-up, pure-Qt media player inspired by SMPlayer, released
under the **GPL-3.0-or-later** license.
