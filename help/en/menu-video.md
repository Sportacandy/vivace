# Video

The **Video** menu adjusts how the picture is shown. These settings apply to
the current file and reset for each new one.

- **Track** — choose among the file's embedded video tracks.
- **Fullscreen** (`F`) — toggle fullscreen; `Esc` leaves it. You can also
  double-click the video.
- **Size** — resize the window to **50% / 100% / 200%** of the video's native
  resolution.
- **Zoom and pan**
  - **Reset** (`Shift+E`), **Zoom −** (`W`), **Zoom +** (`E`).
  - **Move left / right / up / down** (`Alt`+arrow keys) to pan a zoomed image.
- **Aspect ratio** — force a display aspect (Auto, 4:3, 16:9, …).
- **Rotate** — None, 90° clockwise, 90° counterclockwise, or 180°.
- **Flip image** / **Mirror image** — flip vertically or horizontally.
- **Equalizer…** (`Ctrl+E`) — brightness, contrast, hue, saturation and gamma.
  *Set as default values* applies your settings to future files.
- **Screenshot** (`S`) — save the current frame. The folder and image format
  are set in *Preferences ▸ General*.
