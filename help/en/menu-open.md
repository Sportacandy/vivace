# Open

The **Open** menu is how you load media into Vivace.

- **File…** (`Ctrl+O`) — open one or more media files. Selecting several loads
  them as a playlist.
- **Recent files** — reopen something played recently; **Clear** empties the
  list. How many entries are kept is set in *Preferences ▸ Interface ▸ Privacy*.
- **Favorites** — your saved files and streams, organised into submenus. Edit
  them from *Open ▸ Favorites ▸ Edit favorites…*.
- **Directory…** — play every media file in a folder (optionally recursively).
- **Playlist…** — load a saved `.m3u` / `.m3u8` playlist.
- **Disc**
  - **DVD from drive or folder…** — play an unencrypted DVD-Video, including
    its menus. See [Browse](menu-browse.md) for titles and chapters.
  - **Blu-ray**, **Audio CD** — not yet supported.
- **URL…** (`Ctrl+U`) — play a network stream. With YouTube support enabled
  (*Preferences ▸ Network*), YouTube links are handled here too.
- **YouTube cache…** — browse videos Vivace downloaded (when using the
  download-and-play YouTube mode).
- **TV** / **Radio** — your channel lists (imported from SMPlayer if present).
- **Quit** (`Ctrl+Q`) — close Vivace.

**Tip:** you can also drag files, folders or a `VIDEO_TS` folder straight onto
the window.
