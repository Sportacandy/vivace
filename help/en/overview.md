# Getting Started

Welcome to **Vivace** (pronounced *vee-VAH-cheh*), a fast, pure-Qt media
player. This is Vivace's built-in, offline help. Pick a topic from the list on
the left, or start with the basics below.

## Play something

- **A file** — *Open ▸ File…* (`Ctrl+O`), or drag a file onto the window.
- **A folder** — *Open ▸ Directory…* plays everything in a folder.
- **A URL / stream** — *Open ▸ URL…* (`Ctrl+U`).
- **A DVD** — *Open ▸ Disc ▸ DVD…*, or drop a folder containing `VIDEO_TS`.

Dropping several files at once loads them as a playlist.

## Essential shortcuts

| Action | Shortcut |
| --- | --- |
| Play / pause | `Space` |
| Seek back / forward | `←` / `→` |
| Volume down / up | `↓` / `↑` |
| Mute | `M` |
| Fullscreen | `F` (or double-click the video) |
| Exit fullscreen | `Esc` |
| Take a screenshot | `S` |
| Playlist | `F9` |
| Preferences | `Ctrl+P` |

Every shortcut can be changed in *Preferences ▸ Keyboard and mouse*.
