# Browse

The **Browse** menu navigates within the current media — mainly DVDs and
chapters — and manages bookmarks.

- **DVD menu** — return to the disc's menu (enabled while a DVD with menus is
  playing). In a menu, click a button, or use the arrow keys and `Enter`. DVD
  menus are experimental; behaviour is configured in *Preferences ▸ Drives*.
- **Title** — for a DVD, jump to a specific title (with its duration).
- **Chapters** — jump to a chapter. For files, chapters are read from the
  container (mkv/mp4); for DVDs, from the disc. Chapter marks also appear as
  ticks on the seek slider.
- **Bookmarks** — your own named positions in the current file:
  - **Add new bookmark** (`Ctrl+A`) — bookmark the current time.
  - **Edit bookmarks…** — rename or remove them; click one to jump there.

  Bookmarks are stored per file (per title for DVDs) and kept separately from
  the resume history.
