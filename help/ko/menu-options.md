# 옵션

**옵션** 메뉴에는 환경 설정과 인터페이스 구성이 있습니다.

- **환경 설정…**(`Ctrl+P`) — 주 설정 대화상자: 일반, 인터페이스, 자막, 키보드 및
  마우스, 재생 목록, 드라이브, TV, 파일 형식, 업데이트, 네트워크, 고급. 변경 사항은
  즉시 적용되며 **취소** 는 되돌립니다.
  - **네트워크** 에는 OpenSubtitles, YouTube, 프록시, 전송 탭이 있습니다.
    **프록시** 탭에서는 선택적 HTTP 또는 SOCKS5 프록시를 앱 전체에 적용할 수
    있습니다(OpenSubtitles 검색, 업데이트 확인, 그리고 HTTP인 경우에만 미디어
    재생과 yt-dlp). **전송** 탭은 *재생 ▸ 다음으로 전송 ▸ 스마트폰/태블릿* 이
    수신 대기하는 포트를 고정합니다. 이제 OpenSubtitles 계정 비밀번호와
    프록시 비밀번호는 Vivace 자체 설정이 아니라 운영체제의 자격 증명
    관리자에 안전하게 저장됩니다.
- **시스템 트레이에 아이콘 표시** — Vivace를 트레이에서 접근할 수 있게 유지합니다.
- **도구 모음**
  - **도구 모음** / **컨트롤 바** — 각 막대를 표시하거나 숨깁니다.
  - **주 도구 모음 편집…** / **컨트롤 바 편집…** — 표시할 버튼과 순서, 아이콘 크기를 선택합니다.
- **상태 표시줄**
  - **상태 표시줄 표시** 및 표시 내용: **영상 정보**, **오디오 정보**, **형식 정보**,
    **비트레이트 정보**, **프레임 카운터**, **전체 시간 표시**, **남은 시간 표시**,
    **현재 시간을 밀리초로 표시**.

**팁:** 전체 레이아웃(Basic / Mini / MPC)은 *환경 설정 ▸ 인터페이스* 에서 선택합니다.

## YouTube 다운로드용 쿠키 내보내기

**쿠키 파일:** 필드(*환경 설정 ▸ 네트워크 ▸ YouTube*)를 사용하면
**다운로드 후 재생** 및 **외부 도구** YouTube 모드가 마치 로그인한 것처럼
동작하게 할 수 있습니다 — 연령 제한, 회원 전용 또는 그 밖의 계정 인증이
필요한 동영상에 필요하며, 전체 HD/4K 다운로드를 여는 열쇠이기도 합니다.
이 필드에는 일반 텍스트 `cookies.txt` 파일이 필요하며, 고전적인 Netscape
쿠키 저장 형식(yt-dlp 자체의 `--cookies` 옵션이 읽는 것과 동일한 형식)이어야
합니다. Vivace는 브라우저 프로필에서 직접 쿠키를 읽지 않습니다.

**만드는 방법:**

1. 사용하려는 접근 권한을 가진 계정으로 평소 사용하는 브라우저에서
   youtube.com에 로그인합니다.
2. Netscape 형식으로 내보내는 쿠키 내보내기 브라우저 확장 프로그램을
   설치합니다 — Chrome, Edge 또는 Brave의 경우 브라우저 확장 프로그램
   스토어에서 "Get cookies.txt"와 같은 것을 검색하고, Firefox의 경우
   "cookies.txt"를 검색하세요. 고전적인 Netscape `cookies.txt` 형식으로
   내보낸다고 명확히 밝힌 확장 프로그램이라면 무엇이든 사용할 수 있습니다.
3. youtube.com 탭을 연 상태에서 확장 프로그램을 사용하여 해당 사이트의
   쿠키를 내보내고, 결과를 디스크의 어딘가에 `.txt` 파일로 저장합니다.
4. Vivace에서 *환경 설정 ▸ 네트워크 ▸ YouTube* 를 열고 **쿠키 파일:** 옆의
   **찾아보기…** 를 사용하여 해당 파일을 선택합니다.

**참고 사항:**

- `cookies.txt` 파일은 사실상 저장된 로그인 세션입니다 — 이 파일을 가진
  사람은 누구나 쿠키가 만료되거나 로그아웃할 때까지 당신의 YouTube 계정
  처럼 행동할 수 있습니다. 다른 사람이 접근할 수 없는 곳에 보관하고
  공유하지 마세요.
- 쿠키는 **다운로드** 경로(다운로드 후 재생 / 외부 도구)에서만 사용됩니다.
  Vivace는 **스트리밍** 모드에서는 의도적으로 쿠키를 절대 전송하지
  않습니다 — 로그인된 스트림 URL은 Vivace의 단순한 동영상 재생기가 열 수
  없는 방식으로 해당 세션에 묶여 있기 때문에, 쿠키 파일이 설정되어 있어도
  스트리밍은 익명으로 유지됩니다.
- 쿠키는 만료됩니다. 이전에 잘 되던 다운로드가 실패하기 시작하거나 더
  낮은 화질/공개 버전 결과로 대체된다면, 새 `cookies.txt` 를 다시
  내보내세요.
