# 시작하기

순수 Qt로 만든 빠른 미디어 플레이어 **Vivace**(발음: *비-바-체*)에 오신 것을
환영합니다. 이것은 Vivace의 내장 오프라인 도움말입니다. 왼쪽 목록에서 주제를
선택하거나 아래의 기본 사항부터 시작하세요.

## 무언가 재생하기

- **파일** — *열기 ▸ 파일…*(`Ctrl+O`), 또는 창으로 파일을 끌어다 놓습니다.
- **폴더** — *열기 ▸ 디렉터리…* 는 폴더의 모든 것을 재생합니다.
- **URL / 스트림** — *열기 ▸ URL…*(`Ctrl+U`).
- **DVD** — *열기 ▸ 디스크 ▸ DVD…*, 또는 `VIDEO_TS`가 포함된 폴더를 놓습니다.

여러 파일을 한 번에 놓으면 재생 목록으로 불러옵니다.

## 필수 단축키

| 동작 | 단축키 |
| --- | --- |
| 재생 / 일시 정지 | `Space` |
| 뒤로 / 앞으로 탐색 | `←` / `→` |
| 음량 내림 / 올림 | `↓` / `↑` |
| 음소거 | `M` |
| 전체 화면 | `F`(또는 동영상 두 번 클릭) |
| 전체 화면 나가기 | `Esc` |
| 스크린샷 찍기 | `S` |
| 재생 목록 | `F9` |
| 환경 설정 | `Ctrl+P` |

모든 단축키는 *환경 설정 ▸ 키보드 및 마우스* 에서 변경할 수 있습니다.

## 메뉴별 도움말

Vivace의 각 메뉴에는 고유한 페이지가 있습니다:

- [열기](menu-open.md) — 파일, 폴더, 디스크, URL, 즐겨찾기 열기
- [재생](menu-play.md) — 재생, 탐색, 속도, A-B 반복
- [영상](menu-video.md) — 전체 화면, 크기, 확대, 화면 비율, 이퀄라이저, 스크린샷
- [소리](menu-audio.md) — 트랙, 음량, 지연
- [자막](menu-subtitles.md) — 트랙, 외부 파일, 타이밍
- [탐색](menu-browse.md) — DVD 타이틀, 챕터, 책갈피
- [보기](menu-view.md) — 미디어 정보, 재생 목록, OSD
- [옵션](menu-options.md) — 환경 설정, 트레이 아이콘, 도구 모음, 상태 표시줄
- [도움말](menu-help.md) — 이 도움말, 업데이트, 정보
