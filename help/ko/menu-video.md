# 영상

**영상** 메뉴는 화면 표시 방식을 조정합니다. 이 설정은 현재 파일에 적용되며 새
파일마다 재설정됩니다.

- **트랙** — 파일의 내장 영상 트랙 중에서 선택합니다.
- **전체 화면**(`F`) — 전체 화면을 전환하고 `Esc` 로 나갑니다. 동영상을 두 번
  클릭할 수도 있습니다.
- **크기** — 창을 동영상 원본 해상도의 **50% / 100% / 200%** 로 조정합니다.
- **확대 및 이동**
  - **재설정**(`Shift+E`), **축소**(`W`), **확대**(`E`).
  - **왼쪽 / 오른쪽 / 위 / 아래로 이동**(`Alt`+방향키) 하여 확대된 화면을 이동합니다.
- **화면 비율** — 표시 비율을 강제합니다(자동, 4:3, 16:9, …).
- **회전** — 없음, 시계 방향 90°, 반시계 방향 90° 또는 180°.
- **화면 뒤집기** / **화면 좌우 반전** — 세로 또는 가로로 뒤집습니다.
- **이퀄라이저…**(`Ctrl+E`) — 밝기, 대비, 색상, 채도 및 감마.
  *기본값으로 설정* 은 설정을 이후 파일에 적용합니다.
- **스크린샷**(`S`) — 현재 프레임을 저장합니다. 폴더와 이미지 형식은
  *환경 설정 ▸ 일반* 에서 설정합니다.
