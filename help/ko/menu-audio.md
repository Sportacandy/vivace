# 소리

**소리** 메뉴는 소리를 제어합니다.

- **트랙** — 파일의 내장 오디오 트랙 중에서 선택합니다. 선호 언어는
  *환경 설정 ▸ 일반* 에서 자동으로 선택할 수 있습니다.
- **음소거**(`M`) — 소리를 켜고 끕니다.
- **음량 −** / **음량 +**(`↓` / `↑`) — 음량을 변경합니다. 단계 크기는
  *환경 설정 ▸ 일반* 에서 설정합니다.
- **지연 −** / **지연 +** — 입 모양 동기화를 맞추기 위해 오디오를 앞당기거나 늦춥니다.
- **지연 설정…** — 파일별 정확한 오디오 지연을 입력합니다.

## 오디오 지연 정보

파일에 적용되는 지연은 **장치별** 지연(각 오디오 출력 장치에 대해 기억되며 —
블루투스 지연에 유용, *환경 설정 ▸ 일반 ▸ 오디오* 에서 설정)과 이 메뉴의 **파일별**
지연의 합입니다. Vivace는 맞추기 위해 영상을 지연시켜 오프셋을 적용하므로, 음(−)의
지연(오디오가 더 먼저 나와야 함)도 예상대로 작동합니다.
