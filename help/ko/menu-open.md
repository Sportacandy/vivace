# 열기

**열기** 메뉴는 미디어를 Vivace로 불러오는 방법입니다.

- **파일…**(`Ctrl+O`) — 하나 이상의 미디어 파일을 엽니다. 여러 개를 선택하면 재생
  목록으로 불러옵니다.
- **최근 파일** — 최근에 재생한 것을 다시 엽니다. **지우기** 는 목록을 비웁니다.
  보관할 항목 수는 *환경 설정 ▸ 인터페이스 ▸ 개인정보* 에서 설정합니다.
- **즐겨찾기** — 저장한 파일과 스트림을 하위 메뉴로 정리한 것입니다.
  *열기 ▸ 즐겨찾기 ▸ 즐겨찾기 편집…* 에서 편집합니다.
- **디렉터리…** — 폴더의 모든 미디어 파일을 재생합니다(선택적으로 재귀적으로).
- **재생 목록…** — 저장된 `.m3u` / `.m3u8` 재생 목록을 불러옵니다.
- **디스크**
  - **드라이브 또는 폴더에서 DVD…** — 암호화되지 않은 DVD-Video를 메뉴 포함하여
    재생합니다. 타이틀과 챕터는 [탐색](menu-browse.md) 을 참조하세요.
  - **블루레이**, **오디오 CD** — 아직 지원되지 않습니다.
- **URL…**(`Ctrl+U`) — 네트워크 스트림을 재생합니다. YouTube 지원이 켜져 있으면
  (*환경 설정 ▸ 네트워크*) YouTube 링크도 여기서 처리됩니다.
- **YouTube 캐시…** — Vivace가 다운로드한 동영상을 탐색합니다(다운로드 후 재생 모드일 때).
- **TV** / **라디오** — 채널 목록(있으면 SMPlayer에서 가져옴).
- **끝내기**(`Ctrl+Q`) — Vivace를 닫습니다.

**팁:** 파일, 폴더 또는 `VIDEO_TS` 폴더를 창으로 직접 끌어다 놓을 수도 있습니다.
