# 도움말

**도움말** 메뉴.

- **내용**(`F1`) — 이 도움말 창을 엽니다. 왼쪽 목록은 목차입니다. 주제를 클릭하면
  오른쪽에서 읽을 수 있습니다. 페이지 내 링크는 관련 주제를 여기서 열고, 외부 링크는
  웹 브라우저에서 열립니다.
- **업데이트 확인** — 더 새로운 Vivace가 있는지 확인합니다. 자동 확인은
  *환경 설정 ▸ 업데이트* 에서 켤 수 있습니다.
- **YouTube 지원 설치 / 업데이트** — Vivace가 YouTube 링크 재생에 사용하는 도우미
  프로그램 `yt-dlp` 를 다운로드하거나 업데이트합니다(*환경 설정 ▸ 네트워크* 참조).
- **Vivace 정보** — 버전, Qt 버전, 감사의 말, 라이선스.

Vivace는 SMPlayer에서 영감을 받아 처음부터 만든 순수 Qt 미디어 플레이어이며
**GPL-3.0-or-later** 라이선스로 배포됩니다.
