# 자막

**자막** 메뉴는 자막을 관리합니다.

- **트랙** — 내장 자막 트랙을 선택하거나 **끄기** 로 자막을 숨깁니다.
- **자막 열기…** — 외부 `.srt`, `.vtt` 또는 `.ass`/`.ssa` 파일을 불러옵니다.
  Vivace는 동영상 옆에 있는 자막을 자동으로 불러올 수도 있습니다(*환경 설정 ▸ 자막*).
- **OpenSubtitles에서 자막 찾기…** — 온라인으로 자막을 검색하고 다운로드합니다.
  이를 위해서는 *환경 설정 ▸ 네트워크* 에서 입력하는 OpenSubtitles API 키가 필요합니다.
- **자막 내리기** — 현재 불러온 외부 자막을 제거합니다.
- **지연 −** / **지연 +** — 자막 타이밍을 앞당기거나 늦춥니다.
- **지연 설정…** — 정확한 자막 지연을 입력합니다.

자막 글꼴, 크기 및 위치는 *환경 설정 ▸ 자막* 에서 설정합니다.

**참고:** 외부에서 불러온 자막은 *열기* / *내리기* 로 제어합니다 — *트랙*(내장 트랙과
*끄기* 를 표시)에는 나열되지 않습니다.
