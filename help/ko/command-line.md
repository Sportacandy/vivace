# 명령줄 옵션

Vivace는 미디어 파일과 몇 가지 옵션과 함께 터미널에서 시작할 수 있습니다. GNU 스타일의
긴 옵션을 사용하세요(짧은 별칭은 괄호 안에 표시됨).

```
vivace [옵션] [미디어]
```

- **미디어** — 재생할 파일 경로 또는 URL. Vivace가 일반적으로 열 수 있는 모든 것(로컬
  파일, 폴더, `VIDEO_TS`, 네트워크 스트림, YouTube URL)이 여기서 작동합니다.

## 옵션

- **`--help`(`-h`)** — 옵션 목록을 표시하고 종료합니다.
- **`--version`(`-v`)** — 버전을 표시하고 종료합니다.
- **`--fullscreen`(`-f`)** — 전체 화면으로 시작합니다.
- **`--ontop`(`-t`)** — 창을 다른 창 위에 유지합니다.
- **`--close-at-end`(`-c`)** — 재생이 끝나면 Vivace를 닫습니다.
- **`--sub <파일>`(`-s`)** — 지정한 자막 파일을 불러옵니다.
- **`--start <시간>`(`-b`)** — `<시간>` 에서 재생을 시작합니다. `h:m:s`, `m:s` 또는
  초 단위 숫자로 지정합니다.
- **`--pos <x,y>`** — 창을 화면 위치 `x,y` 에 배치합니다.
- **`--size <w,h>`** — 창 크기를 `w`×`h` 로 설정합니다.

## 예제

```
vivace movie.mkv
vivace --fullscreen --start 1:30 movie.mkv
vivace -f -b 90 -s movie.en.srt movie.mkv
vivace --pos 100,100 --size 1280,720 movie.mp4
```

**참고(Windows):** Vivace는 GUI 애플리케이션이므로 `--help` / `--version` 은 실행한
터미널로 출력됩니다(셸 프롬프트가 먼저 돌아올 수 있어 텍스트가 그 뒤에 나타날 수
있음). 단일 인스턴스 모드가 켜져 있으면(*환경 설정 ▸ 인터페이스 ▸ 인스턴스*), 미디어
파일과 이 옵션들로 다시 시작하면 이미 실행 중인 창에 적용됩니다.
