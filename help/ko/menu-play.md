# 재생

**재생** 메뉴는 재생과 탐색을 제어합니다.

## 조작

- **재생**, **일시 정지**, **정지** — `Space` 는 재생/일시 정지를 전환하고
  `Ctrl+Space` 는 정지합니다.
- **다음 프레임**(`.`) 및 **이전 프레임** — 한 프레임 앞으로 또는 뒤로 이동합니다.
- **이전** / **다음** — 재생 목록 항목 사이를 이동합니다.

## 탐색

양방향으로 세 가지 탐색 간격(**뒤로** / **앞으로**)이 제공되며 현재 길이와 함께
표시됩니다. 간격은 *환경 설정 ▸ 인터페이스 ▸ 탐색* 에서 조정합니다. `←` / `→`,
탐색 슬라이더 또는 동영상 위에서 마우스 휠로도 탐색할 수 있습니다.

## 속도

- **일반 속도**(`Backspace`), **속도 절반**, **속도 두 배**.
- **속도 −10% / +10%** 및 더 세밀한 ±4% / ±1% 단계.
- **음정 보정** — 비정상 속도에서도 음성을 자연스럽게 유지합니다.

재생 속도는 파일별이며 새 파일마다 일반으로 재설정됩니다.

## A-B 구간

파일의 일부를 반복합니다:

- **A 마커 설정**, **B 마커 설정** — 시작과 끝을 표시합니다.
- **반복** — A-B 구간을 반복합니다(이 항목이 켜져 있을 때만 구간이 반복됩니다).
- **A-B 마커 지우기** — 마커를 제거합니다. 마커는 새 파일마다 재설정됩니다.

**이동…**(특정 시간으로 이동)은 이후 릴리스에 계획되어 있습니다.

## 전송

**다음으로 전송 ▸ 스마트폰/태블릿…** 은 작은 내장 웹 서버를 시작하여 같은
네트워크의 휴대폰이나 태블릿이 페이지를 열고 Vivace가 현재 재생 중인 파일을
재생할 수 있게 합니다 — QR 코드를 스캔하거나 표시된 주소를 입력하면 됩니다.
포트는 *환경 설정 ▸ 네트워크 ▸ 전송* 에 고정되어 있으므로(세션마다 편집할 수
없음) 방화벽/라우터에서 한 번만 허용하면 됩니다. 현재 재생 중인 파일만
전송되며, 전송이 켜져 있는 동안에만 작동합니다.
