# 탐색

**탐색** 메뉴는 현재 미디어 내를 이동하며 — 주로 DVD와 챕터 — 책갈피를 관리합니다.

- **DVD 메뉴** — 디스크의 메뉴로 돌아갑니다(메뉴가 있는 DVD가 재생 중일 때 사용
  가능). 메뉴에서 버튼을 클릭하거나 방향키와 `Enter` 를 사용합니다. DVD 메뉴는
  실험적 기능입니다. 동작은 *환경 설정 ▸ 드라이브* 에서 구성합니다.
- **타이틀** — DVD의 경우 특정 타이틀로 이동합니다(길이 표시).
- **챕터** — 챕터로 이동합니다. 파일의 경우 컨테이너(mkv/mp4)에서, DVD의 경우
  디스크에서 챕터를 읽습니다. 챕터 표시는 탐색 슬라이더에도 눈금으로 나타납니다.
- **책갈피** — 현재 파일에서 직접 이름 붙인 위치:
  - **책갈피 추가**(`Ctrl+A`) — 현재 시간에 책갈피를 만듭니다.
  - **책갈피 편집…** — 이름을 바꾸거나 제거합니다. 하나를 클릭하면 그곳으로 이동합니다.

  책갈피는 파일별(DVD는 타이틀별)로 저장되며 이어보기 기록과 별도로 보관됩니다.
