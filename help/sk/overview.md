# Začíname

Vitajte vo **Vivace** (vyslovuje sa *ví-VA-če*), rýchlom prehrávači médií čisto v
Qt. Toto je vstavaný offline pomocník Vivace. Vyberte tému zo zoznamu vľavo alebo
začnite základmi nižšie.

## Niečo prehrať

- **Súbor** — *Otvoriť ▸ Súbor…* (`Ctrl+O`), alebo presuňte súbor na okno.
- **Priečinok** — *Otvoriť ▸ Adresár…* prehrá všetko v priečinku.
- **URL / stream** — *Otvoriť ▸ URL…* (`Ctrl+U`).
- **DVD** — *Otvoriť ▸ Disk ▸ DVD…*, alebo presuňte priečinok obsahujúci `VIDEO_TS`.

Presunutie viacerých súborov naraz ich načíta ako zoznam skladieb.

## Základné klávesové skratky

| Akcia | Skratka |
| --- | --- |
| Prehrať / pozastaviť | `Medzerník` |
| Späť / vpred | `←` / `→` |
| Stíšiť / zosilniť | `↓` / `↑` |
| Stlmiť zvuk | `M` |
| Celá obrazovka | `F` (alebo dvojklik na video) |
| Opustiť celú obrazovku | `Esc` |
| Urobiť snímku obrazovky | `S` |
| Zoznam skladieb | `F9` |
| Nastavenia | `Ctrl+P` |

Každú skratku možno zmeniť v *Nastavenia ▸ Klávesnica a myš*.

## Pomocník podľa ponuky

Každá ponuka Vivace má vlastnú stránku:

- [Otvoriť](menu-open.md) — otváranie súborov, priečinkov, diskov, URL, obľúbených
- [Prehrávanie](menu-play.md) — prehrávanie, prevíjanie, rýchlosť, opakovanie A-B
- [Video](menu-video.md) — celá obrazovka, veľkosť, priblíženie, pomer strán, ekvalizér, snímky
- [Zvuk](menu-audio.md) — stopy, hlasitosť, oneskorenie
- [Titulky](menu-subtitles.md) — stopy, externé súbory, časovanie
- [Prehľadávať](menu-browse.md) — tituly DVD, kapitoly, záložky
- [Zobraziť](menu-view.md) — informácie o médiu, zoznam skladieb, OSD
- [Možnosti](menu-options.md) — nastavenia, ikona v oblasti oznámení, panely nástrojov, stavový riadok
- [Pomocník](menu-help.md) — tento pomocník, aktualizácie, o programe
