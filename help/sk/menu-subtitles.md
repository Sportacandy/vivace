# Titulky

Ponuka **Titulky** spravuje titulky.

- **Stopa** — vybrať vloženú stopu titulkov alebo **Vypnuté** na ich skrytie.
- **Načítať titulky…** — načítať externý súbor `.srt`, `.vtt` alebo `.ass`/`.ssa`.
  Vivace môže tiež automaticky načítať titulky ležiace vedľa videa
  (*Nastavenia ▸ Titulky*).
- **Nájsť titulky na OpenSubtitles…** — hľadať a sťahovať titulky online.
  Vyžaduje kľúč API OpenSubtitles zadaný v *Nastavenia ▸ Sieť*.
- **Uvoľniť titulky** — odstrániť aktuálne načítané externé titulky.
- **Oneskorenie −** / **Oneskorenie +** — posunúť časovanie titulkov skôr alebo neskôr.
- **Nastaviť oneskorenie…** — zadať presné oneskorenie titulkov.

Písmo, veľkosť a poloha titulkov sa nastavujú v *Nastavenia ▸ Titulky*.

**Poznámka:** externe načítané titulky sa ovládajú cez *Načítať* / *Uvoľniť* —
nie sú uvedené v *Stopa* (kde sú vložené stopy a *Vypnuté*).
