# Вигляд

Меню **Вигляд** показує відомості та додаткові панелі.

- **Відомості та властивості…** — докладне вікно про поточне медіа: загальні
  відомості, демультиплексор і відео-/аудіокодеки (з виділенням форматів, які
  підтримує рушій Vivace). Лише для інформації.
- **Список відтворення** (`F9`) — показати або сховати список відтворення. Залежно
  від *Налаштування ▸ Інтерфейс* він показується як закріплена панель або окреме
  вікно. Див. [Відкрити](menu-open.md) для завантаження списків відтворення.
- **OSD** — екранне відображення над відео:
  - рівень деталізації (лише субтитри; гучність + перемотування; гучність +
    перемотування + таймер) розширюється; тривалість OSD та розмір шрифту — у
    *Налаштування ▸ Загальні*.
