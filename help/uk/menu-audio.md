# Звук

Меню **Звук** керує звуком.

- **Доріжка** — вибрати одну з вбудованих звукових доріжок файлу. Бажану мову
  можна вибрати автоматично в *Налаштування ▸ Загальні*.
- **Вимкнути звук** (`M`) — увімкнути/вимкнути звук.
- **Тихіше** / **Гучніше** (`↓` / `↑`) — змінити гучність; розмір кроку задають у
  *Налаштування ▸ Загальні*.
- **Затримка −** / **Затримка +** — зсунути звук раніше або пізніше для виправлення синхронізації губ.
- **Задати затримку…** — увести точну затримку звуку для файлу.

## Про затримку звуку

Затримка, застосована до файлу, — це сума затримки **для пристрою**
(запам'ятовується для кожного пристрою виведення звуку — корисно за затримки
Bluetooth, задається в *Налаштування ▸ Загальні ▸ Аудіо*) і затримки **для
файлу** з цього меню. Vivace застосовує зсув, затримуючи відео для збігу, тож
від'ємні затримки (звук має йти раніше) працюють як очікується.
