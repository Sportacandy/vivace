# Субтитри

Меню **Субтитри** керує субтитрами.

- **Доріжка** — вибрати вбудовану доріжку субтитрів або **Вимкнено**, щоб їх приховати.
- **Завантажити субтитри…** — завантажити зовнішній файл `.srt`, `.vtt` або
  `.ass`/`.ssa`. Vivace також може автоматично завантажити субтитри поряд із відео
  (*Налаштування ▸ Субтитри*).
- **Знайти субтитри на OpenSubtitles…** — шукати й завантажувати субтитри онлайн.
  Потрібен ключ API OpenSubtitles, який вводять у *Налаштування ▸ Мережа*.
- **Вивантажити субтитри** — прибрати поточні завантажені зовнішні субтитри.
- **Затримка −** / **Затримка +** — зсунути синхронізацію субтитрів раніше або пізніше.
- **Задати затримку…** — увести точну затримку субтитрів.

Шрифт, розмір і положення субтитрів задають у *Налаштування ▸ Субтитри*.

**Примітка:** зовні завантажені субтитри керуються через *Завантажити* /
*Вивантажити* — вони не показані в *Доріжка* (де показано вбудовані доріжки і *Вимкнено*).
