# Огляд

Меню **Огляд** переміщується в межах поточного медіа — переважно DVD і розділи —
та керує закладками.

- **Меню DVD** — повернутися до меню диска (доступно, поки відтворюється DVD з
  меню). У меню клацніть кнопку або скористайтеся стрілками та `Enter`. Меню DVD
  експериментальні; поведінку налаштовують у *Налаштування ▸ Приводи*.
- **Заголовок** — для DVD перейти до певного заголовка (з його тривалістю).
- **Розділи** — перейти до розділу. Для файлів розділи читаються з контейнера
  (mkv/mp4); для DVD — з диска. Позначки розділів також показуються рисками на
  повзунку перемотування.
- **Закладки** — ваші іменовані позиції в поточному файлі:
  - **Додати закладку** (`Ctrl+A`) — зробити закладку на поточному часі.
  - **Змінити закладки…** — перейменувати або вилучити їх; клацніть по одній, щоб перейти туди.

  Закладки зберігаються для кожного файлу (для DVD — для кожного заголовка)
  окремо від історії відновлення.
