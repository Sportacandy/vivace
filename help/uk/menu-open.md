# Відкрити

Меню **Відкрити** — це спосіб завантажити медіа у Vivace.

- **Файл…** (`Ctrl+O`) — відкрити один або кілька медіафайлів. Вибір кількох
  завантажує їх як список відтворення.
- **Нещодавні файли** — знову відкрити щось нещодавно відтворене; **Очистити**
  спорожнює список. Скільки записів зберігається, задається в
  *Налаштування ▸ Інтерфейс ▸ Конфіденційність*.
- **Обране** — збережені файли та потоки, впорядковані в підменю. Змінюйте їх з
  *Відкрити ▸ Обране ▸ Змінити обране…*.
- **Каталог…** — відтворити кожен медіафайл у теці (за бажанням рекурсивно).
- **Список відтворення…** — завантажити збережений список `.m3u` / `.m3u8`.
- **Диск**
  - **DVD з привода або теки…** — відтворити незашифроване DVD-Video разом із його
    меню. Див. [Огляд](menu-browse.md) для заголовків і розділів.
  - **Blu-ray**, **Аудіо-CD** — поки не підтримуються.
- **URL…** (`Ctrl+U`) — відтворити мережевий потік. З увімкненою підтримкою
  YouTube (*Налаштування ▸ Мережа*) посилання YouTube теж обробляються тут.
- **Кеш YouTube…** — переглянути завантажені Vivace відео (у режимі завантаження й відтворення).
- **ТБ** / **Радіо** — ваші списки каналів (імпортовані з SMPlayer, якщо є).
- **Вийти** (`Ctrl+Q`) — закрити Vivace.

**Порада:** файли, теки або теку `VIDEO_TS` можна також перетягнути прямо у вікно.
