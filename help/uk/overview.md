# Початок роботи

Ласкаво просимо до **Vivace** (вимовляється *ві-ВА-че*) — швидкого медіапрогравача
повністю на Qt. Це вбудована автономна довідка Vivace. Виберіть тему зі списку
ліворуч або почніть з основ нижче.

## Відтворити щось

- **Файл** — *Відкрити ▸ Файл…* (`Ctrl+O`) або перетягніть файл у вікно.
- **Теку** — *Відкрити ▸ Каталог…* відтворює весь вміст теки.
- **URL / потік** — *Відкрити ▸ URL…* (`Ctrl+U`).
- **DVD** — *Відкрити ▸ Диск ▸ DVD…* або перетягніть теку з `VIDEO_TS`.

Перетягування кількох файлів одразу завантажує їх як список відтворення.

## Основні комбінації клавіш

| Дія | Комбінація |
| --- | --- |
| Відтворення / пауза | `Пробіл` |
| Назад / вперед | `←` / `→` |
| Тихіше / гучніше | `↓` / `↑` |
| Вимкнути звук | `M` |
| Повний екран | `F` (або подвійний клац по відео) |
| Вийти з повного екрана | `Esc` |
| Зробити знімок екрана | `S` |
| Список відтворення | `F9` |
| Налаштування | `Ctrl+P` |

Будь-яку комбінацію можна змінити в *Налаштування ▸ Клавіатура та миша*.

Кожне меню Vivace має власну сторінку:

- [Відео](menu-video.md) — повний екран, розмір, масштаб, співвідношення сторін, еквалайзер, знімки

- [Субтитри](menu-subtitles.md) — доріжки, зовнішні файли, синхронізація
- [Огляд](menu-browse.md) — заголовки DVD, розділи, закладки
- [Вигляд](menu-view.md) — відомості про медіа, список відтворення, OSD
- [Параметри](menu-options.md) — налаштування, значок у лотку, панелі інструментів, рядок стану
