# Параметри командного рядка

Vivace можна запустити з термінала з медіафайлом і кількома параметрами.
Використовуйте довгі параметри у стилі GNU (короткі псевдоніми вказані в дужках).

```
vivace [параметри] [медіа]
```

- **медіа** — шлях до файлу або URL для відтворення. Тут працює все, що Vivace
  зазвичай може відкрити (локальний файл, тека, `VIDEO_TS`, мережевий потік, URL YouTube).

## Параметри

- **`--help` (`-h`)** — показати список параметрів і вийти.
- **`--version` (`-v`)** — показати версію і вийти.
- **`--fullscreen` (`-f`)** — запустити в повноекранному режимі.
- **`--ontop` (`-t`)** — тримати вікно над іншими вікнами.
- **`--close-at-end` (`-c`)** — закрити Vivace після завершення відтворення.
- **`--sub <файл>` (`-s`)** — завантажити вказаний файл субтитрів.
- **`--start <час>` (`-b`)** — почати відтворення з `<час>`, заданого як `h:m:s`,
  `m:s` або кількість секунд.
- **`--pos <x,y>`** — розмістити вікно в позиції екрана `x,y`.
- **`--size <ш,в>`** — задати розмір вікна `ш`×`в`.

## Приклади

```
vivace movie.mkv
vivace --fullscreen --start 1:30 movie.mkv
vivace -f -b 90 -s movie.en.srt movie.mkv
vivace --pos 100,100 --size 1280,720 movie.mp4
```

**Примітка (Windows):** Vivace — застосунок із графічним інтерфейсом, тож
`--help` / `--version` виводяться в термінал, з якого ви його запустили (запит
оболонки може повернутися першим, тож текст може з'явитися після нього). Коли
ввімкнено режим одного екземпляра (*Налаштування ▸ Інтерфейс ▸ Екземпляри*),
повторний запуск із медіафайлом і цими параметрами застосовує їх до вже
запущеного вікна.
