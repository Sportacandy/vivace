# Відео

Меню **Відео** налаштовує відображення зображення. Ці налаштування застосовуються
до поточного файлу й скидаються для кожного нового.

- **Доріжка** — вибрати одну з вбудованих відеодоріжок файлу.
- **Повний екран** (`F`) — перемкнути повний екран; `Esc` виходить із нього. Можна
  також двічі клацнути по відео.
- **Розмір** — змінити розмір вікна до **50 % / 100 % / 200 %** власної роздільної
  здатності відео.
- **Масштаб і панорама**
  - **Скинути** (`Shift+E`), **Зменшити** (`W`), **Збільшити** (`E`).
  - **Пересунути ліворуч / праворуч / вгору / вниз** (`Alt`+стрілки), щоб
    пересунути збільшене зображення.
- **Співвідношення сторін** — примусово задати співвідношення (Авто, 4:3, 16:9, …).
- **Обертання** — Немає, 90° за годинниковою, 90° проти годинникової або 180°.
- **Віддзеркалити по вертикалі** / **Віддзеркалити по горизонталі** — перевернути зображення.
- **Еквалайзер…** (`Ctrl+E`) — яскравість, контраст, відтінок, насиченість і гама.
  *Установити як типові значення* застосовує ваші налаштування до майбутніх файлів.
- **Знімок екрана** (`S`) — зберегти поточний кадр. Теку та формат зображення
  задають у *Налаштування ▸ Загальні*.
