# Premiers pas

Bienvenue dans **Vivace** (prononcé *vi-VA-tché*), un lecteur multimédia rapide
et entièrement en Qt. Ceci est l'aide hors ligne intégrée de Vivace. Choisissez
un sujet dans la liste de gauche, ou commencez par les bases ci-dessous.

## Lire quelque chose

- **Un fichier** — *Ouvrir ▸ Fichier…* (`Ctrl+O`), ou glissez un fichier sur la fenêtre.
- **Un dossier** — *Ouvrir ▸ Répertoire…* lit tout le contenu d'un dossier.
- **Une URL / un flux** — *Ouvrir ▸ URL…* (`Ctrl+U`).
- **Un DVD** — *Ouvrir ▸ Disque ▸ DVD…*, ou déposez un dossier contenant `VIDEO_TS`.

Déposer plusieurs fichiers à la fois les charge comme une liste de lecture.

## Raccourcis essentiels

| Action | Raccourci |
| --- | --- |
| Lecture / pause | `Espace` |
| Reculer / avancer | `←` / `→` |
| Baisser / monter le volume | `↓` / `↑` |
| Muet | `M` |
| Plein écran | `F` (ou double-clic sur la vidéo) |
| Quitter le plein écran | `Échap` |
| Faire une capture d'écran | `S` |
| Liste de lecture | `F9` |
| Préférences | `Ctrl+P` |

Chaque raccourci peut être modifié dans *Préférences ▸ Clavier et souris*.
