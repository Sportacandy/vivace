# Audio

Le menu **Audio** contrôle le son.

- **Piste** — choisir parmi les pistes audio intégrées du fichier. Une langue
  préférée peut être sélectionnée automatiquement dans *Préférences ▸ Général*.
- **Muet** (`M`) — activer/désactiver le son.
- **Volume −** / **Volume +** (`↓` / `↑`) — changer le volume ; le pas se règle
  dans *Préférences ▸ Général*.
- **Décalage −** / **Décalage +** — avancer ou retarder l'audio pour corriger la synchronisation labiale.
- **Définir le décalage…** — saisir un décalage audio précis par fichier.

## À propos du décalage audio

Le décalage appliqué à un fichier est la somme d'un décalage **par périphérique**
(mémorisé pour chaque périphérique de sortie audio — utile pour la latence
Bluetooth, réglé dans *Préférences ▸ Général ▸ Audio*) et du décalage **par
fichier** de ce menu. Vivace applique le décalage en retardant la vidéo pour
correspondre, de sorte que les décalages négatifs (l'audio doit venir plus tôt)
fonctionnent comme prévu.
