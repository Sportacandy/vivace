# Lecture

Le menu **Lecture** contrôle la lecture et la navigation.

## Transport

- **Lecture**, **Pause**, **Stop** — `Espace` bascule lecture/pause ;
  `Ctrl+Espace` arrête.
- **Image suivante** (`.`) et **Image précédente** — avancer ou reculer d'une image.
- **Précédent** / **Suivant** — se déplacer entre les éléments de la liste de lecture.

## Navigation

Trois pas de navigation sont proposés dans les deux sens (**Reculer** /
**Avancer**), affichés avec leur durée actuelle. Ajustez les pas dans
*Préférences ▸ Interface ▸ Navigation*. Vous pouvez aussi naviguer avec `←` /
`→`, le curseur de navigation ou la molette de la souris sur la vidéo.

## Vitesse

- **Vitesse normale** (`Retour arrière`), **Vitesse de moitié**, **Vitesse double**.
- **Vitesse −10 % / +10 %** et des pas plus fins de ±4 % / ±1 %.
- **Compensation de hauteur tonale** — garde les voix naturelles à vitesse non normale.

La vitesse de lecture est propre à chaque fichier et revient à la normale pour
chaque nouveau fichier.

## Section A-B

Répéter une partie d'un fichier :

- **Définir le repère A**, **Définir le repère B** — marquer le début et la fin.
- **Répéter** — boucler la région A-B (la section ne se répète que si ceci est activé).
- **Effacer les repères A-B** — les supprimer. Les repères sont réinitialisés pour
  chaque nouveau fichier.

**Aller à…** (atteindre un instant précis) est prévu pour une version ultérieure.

## Diffusion

**Diffuser vers ▸ Smartphone/tablette…** démarre un petit serveur web intégré
pour qu'un smartphone ou une tablette sur le même réseau puisse ouvrir une
page et lire le fichier que Vivace lit actuellement — en scannant le code QR
ou en saisissant l'adresse affichée. Le port est fixé dans *Préférences ▸
Réseau ▸ Diffusion* (non modifiable à chaque session), afin de pouvoir
l'autoriser une seule fois dans votre pare-feu/routeur. Seul le fichier en
cours de lecture est diffusé, et uniquement tant que la diffusion est active.
