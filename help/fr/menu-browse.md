# Naviguer

Le menu **Naviguer** se déplace au sein du média en cours — principalement les
DVD et les chapitres — et gère les signets.

- **Menu DVD** — revenir au menu du disque (activé quand un DVD avec menus est en
  lecture). Dans un menu, cliquez sur un bouton, ou utilisez les touches fléchées
  et `Entrée`. Les menus DVD sont expérimentaux ; le comportement se configure
  dans *Préférences ▸ Lecteurs*.
- **Titre** — pour un DVD, aller à un titre précis (avec sa durée).
- **Chapitres** — aller à un chapitre. Pour les fichiers, les chapitres sont lus
  depuis le conteneur (mkv/mp4) ; pour les DVD, depuis le disque. Les marques de
  chapitre apparaissent aussi comme des repères sur le curseur de navigation.
- **Signets** — vos propres positions nommées dans le fichier en cours :
  - **Ajouter un signet** (`Ctrl+A`) — marquer l'instant actuel.
  - **Modifier les signets…** — les renommer ou les supprimer ; cliquez sur l'un
    d'eux pour y aller.

  Les signets sont enregistrés par fichier (par titre pour les DVD) et conservés
  séparément de l'historique de reprise.
