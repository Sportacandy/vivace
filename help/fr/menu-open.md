# Ouvrir

Le menu **Ouvrir** vous permet de charger des médias dans Vivace.

- **Fichier…** (`Ctrl+O`) — ouvrir un ou plusieurs fichiers multimédias. En
  sélectionner plusieurs les charge comme une liste de lecture.
- **Fichiers récents** — rouvrir un élément lu récemment ; **Effacer** vide la
  liste. Le nombre d'entrées conservées se règle dans
  *Préférences ▸ Interface ▸ Confidentialité*.
- **Favoris** — vos fichiers et flux enregistrés, organisés en sous-menus.
  Modifiez-les depuis *Ouvrir ▸ Favoris ▸ Modifier les favoris…*.
- **Répertoire…** — lire tous les fichiers multimédias d'un dossier
  (éventuellement de façon récursive).
- **Liste de lecture…** — charger une liste de lecture `.m3u` / `.m3u8` enregistrée.
- **Disque**
  - **DVD depuis un lecteur ou un dossier…** — lire un DVD-Vidéo non chiffré, y
    compris ses menus. Voir [Naviguer](menu-browse.md) pour les titres et chapitres.
  - **Blu-ray**, **CD audio** — pas encore pris en charge.
- **URL…** (`Ctrl+U`) — lire un flux réseau. Avec la prise en charge de YouTube
  activée (*Préférences ▸ Réseau*), les liens YouTube sont aussi gérés ici.
- **Cache YouTube…** — parcourir les vidéos que Vivace a téléchargées (en mode
  téléchargement puis lecture).
- **TV** / **Radio** — vos listes de chaînes (importées de SMPlayer le cas échéant).
- **Quitter** (`Ctrl+Q`) — fermer Vivace.

**Astuce :** vous pouvez aussi glisser des fichiers, des dossiers ou un dossier
`VIDEO_TS` directement sur la fenêtre.
