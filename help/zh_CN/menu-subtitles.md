# 字幕

**字幕** 菜单管理字幕。

- **音轨** — 选择内嵌字幕轨，或选择 **关闭** 以隐藏字幕。
- **加载字幕…** — 加载外部 `.srt`、`.vtt` 或 `.ass`/`.ssa` 文件。Vivace 也可以自动
  加载视频旁边的字幕（*首选项 ▸ 字幕*）。
- **在 OpenSubtitles 查找字幕…** — 在线搜索并下载字幕。这需要一个 OpenSubtitles
  API 密钥，在 *首选项 ▸ 网络* 中输入。
- **卸载字幕** — 移除当前加载的外部字幕。
- **延迟 −** / **延迟 +** — 提前或延后字幕的时间轴。
- **设置延迟…** — 输入精确的字幕延迟。

字幕的字体、大小和位置在 *首选项 ▸ 字幕* 中设置。

**注意：** 外部加载的字幕通过 *加载* / *卸载* 控制 — 它不会列在 *音轨* 下（那里显示
内嵌音轨和 *关闭*）。
