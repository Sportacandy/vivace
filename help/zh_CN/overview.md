# 入门

欢迎使用 **Vivace**（读作 *vee-VAH-cheh*），一款纯 Qt 打造的快速媒体播放器。这是
Vivace 内置的离线帮助。请从左侧列表中选择一个主题，或先从下面的基础内容开始。

## 播放内容

- **文件** — *打开 ▸ 文件…*（`Ctrl+O`），或将文件拖到窗口上。
- **文件夹** — *打开 ▸ 目录…* 播放文件夹中的所有内容。
- **URL / 流** — *打开 ▸ URL…*（`Ctrl+U`）。
- **DVD** — *打开 ▸ 光盘 ▸ DVD…*，或拖入包含 `VIDEO_TS` 的文件夹。

一次拖入多个文件会将它们作为播放列表加载。

## 基本快捷键

| 操作 | 快捷键 |
| --- | --- |
| 播放 / 暂停 | `空格` |
| 后退 / 前进 | `←` / `→` |
| 减小 / 增大音量 | `↓` / `↑` |
| 静音 | `M` |
| 全屏 | `F`（或双击视频） |
| 退出全屏 | `Esc` |
| 截取屏幕 | `S` |
| 播放列表 | `F9` |
| 首选项 | `Ctrl+P` |

每个快捷键都可以在 *首选项 ▸ 键盘和鼠标* 中更改。
