# 视频

**视频** 菜单调整画面的显示方式。这些设置应用于当前文件，并为每个新文件重置。

- **音轨** — 在文件的内嵌视频轨中选择。
- **全屏**（`F`）— 切换全屏；`Esc` 退出。您也可以双击视频。
- **尺寸** — 将窗口调整为视频原始分辨率的 **50% / 100% / 200%**。
- **缩放和平移**
  - **重置**（`Shift+E`）、**缩小**（`W`）、**放大**（`E`）。
  - **向左 / 右 / 上 / 下移动**（`Alt`+方向键）以平移放大的画面。
- **宽高比** — 强制指定显示宽高比（自动、4:3、16:9……）。
- **旋转** — 无、顺时针 90°、逆时针 90° 或 180°。
- **翻转画面** / **镜像画面** — 垂直或水平翻转。
- **均衡器…**（`Ctrl+E`）— 亮度、对比度、色相、饱和度和伽马。
  *设为默认值* 会将您的设置应用于以后的文件。
- **截图**（`S`）— 保存当前帧。文件夹和图像格式在 *首选项 ▸ 常规* 中设置。
