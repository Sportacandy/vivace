# 打开

**打开** 菜单用于将媒体加载到 Vivace 中。

- **文件…**（`Ctrl+O`）— 打开一个或多个媒体文件。选择多个会将它们作为播放列表加载。
- **最近的文件** — 重新打开最近播放的内容；**清除** 会清空列表。保留多少条目可在
  *首选项 ▸ 界面 ▸ 隐私* 中设置。
- **收藏夹** — 您保存的文件和流，整理为子菜单。可从
  *打开 ▸ 收藏夹 ▸ 编辑收藏夹…* 编辑它们。
- **目录…** — 播放文件夹中的每个媒体文件（可选择递归）。
- **播放列表…** — 加载已保存的 `.m3u` / `.m3u8` 播放列表。
- **光盘**
  - **从驱动器或文件夹打开 DVD…** — 播放未加密的 DVD-Video，包括其菜单。有关标题和
    章节，请参阅 [浏览](menu-browse.md)。
  - **蓝光**、**音频 CD** — 尚不支持。
- **URL…**（`Ctrl+U`）— 播放网络流。启用 YouTube 支持后（*首选项 ▸ 网络*），此处也会
  处理 YouTube 链接。
- **YouTube 缓存…** — 浏览 Vivace 下载的视频（使用下载并播放模式时）。
- **电视** / **广播** — 您的频道列表（若存在，则从 SMPlayer 导入）。
- **退出**（`Ctrl+Q`）— 关闭 Vivace。

**提示：** 您也可以将文件、文件夹或 `VIDEO_TS` 文件夹直接拖到窗口上。
