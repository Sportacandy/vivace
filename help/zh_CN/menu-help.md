# 帮助

**帮助** 菜单。

- **内容**（`F1`）— 打开此帮助窗口。左侧列表是目录；单击某个主题即可在右侧阅读。页面内
  的链接会在此处打开相关主题；外部链接会在您的网页浏览器中打开。
- **检查更新** — 查看是否有更新的 Vivace 版本。可在 *首选项 ▸ 更新* 中启用自动检查。
- **安装 / 更新 YouTube 支持** — 下载或更新 `yt-dlp`，这是 Vivace 用来播放 YouTube
  链接的辅助程序（见 *首选项 ▸ 网络*）。
- **关于 Vivace** — 版本、Qt 版本、致谢和许可。

Vivace 是一款从零构建、纯 Qt 的媒体播放器，受 SMPlayer 启发，在
**GPL-3.0-or-later** 许可下发布。
