# Kom igång

Välkommen till **Vivace** (uttalas *vi-VA-che*), en snabb mediaspelare helt i Qt.
Det här är Vivaces inbyggda offlinehjälp. Välj ett ämne i listan till vänster,
eller börja med grunderna nedan.

## Spela upp något

- **En fil** — *Öppna ▸ Fil…* (`Ctrl+O`), eller dra en fil till fönstret.
- **En mapp** — *Öppna ▸ Katalog…* spelar upp allt i en mapp.
- **En URL / ström** — *Öppna ▸ URL…* (`Ctrl+U`).
- **En dvd** — *Öppna ▸ Skiva ▸ Dvd…*, eller släpp en mapp med `VIDEO_TS`.

Att släppa flera filer samtidigt laddar dem som en spellista.

## Viktiga kortkommandon

| Åtgärd | Kortkommando |
| --- | --- |
| Spela upp / pausa | `Blanksteg` |
| Bakåt / framåt | `←` / `→` |
| Sänk / höj volym | `↓` / `↑` |
| Tysta | `M` |
| Helskärm | `F` (eller dubbelklicka på videon) |
| Lämna helskärm | `Esc` |
| Ta en skärmbild | `S` |
| Spellista | `F9` |
| Inställningar | `Ctrl+P` |

Varje kortkommando kan ändras i *Inställningar ▸ Tangentbord och mus*.

## Hjälp per meny

Varje meny i Vivace har en egen sida:

- [Öppna](menu-open.md) — öppna filer, mappar, skivor, URL:er, favoriter
- [Spela upp](menu-play.md) — uppspelning, spolning, hastighet, A-B-upprepning
- [Video](menu-video.md) — helskärm, storlek, zoom, bildförhållande, equalizer, skärmbilder
- [Ljud](menu-audio.md) — spår, volym, fördröjning
- [Undertexter](menu-subtitles.md) — spår, externa filer, tidsinställning
- [Bläddra](menu-browse.md) — dvd-titlar, kapitel, bokmärken
- [Visa](menu-view.md) — mediainfo, spellista, OSD
- [Alternativ](menu-options.md) — inställningar, aktivitetsfältsikon, verktygsfält, statusfält
- [Hjälp](menu-help.md) — den här hjälpen, uppdateringar, om
