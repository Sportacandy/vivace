# Aloittaminen

Tervetuloa käyttämään **Vivacea** (lausutaan *vii-VA-tše*), nopeaa, täysin
Qt-pohjaista mediasoitinta. Tämä on Vivacen sisäänrakennettu offline-ohje. Valitse
aihe vasemmalla olevasta luettelosta tai aloita alla olevista perusasioista.

## Toista jotain

- **Tiedosto** — *Avaa ▸ Tiedosto…* (`Ctrl+O`), tai vedä tiedosto ikkunaan.
- **Kansio** — *Avaa ▸ Hakemisto…* toistaa kaiken kansiossa.
- **URL / suoratoisto** — *Avaa ▸ URL…* (`Ctrl+U`).
- **DVD** — *Avaa ▸ Levy ▸ DVD…*, tai pudota kansio, joka sisältää `VIDEO_TS`.

Usean tiedoston pudottaminen kerralla lataa ne soittolistaksi.

## Tärkeät pikanäppäimet

| Toiminto | Pikanäppäin |
| --- | --- |
| Toista / keskeytä | `Välilyönti` |
| Taakse / eteen | `←` / `→` |
| Hiljemmalle / kovemmalle | `↓` / `↑` |
| Mykistä | `M` |
| Koko näyttö | `F` (tai kaksoisnapsauta videota) |
| Poistu koko näytöstä | `Esc` |
| Ota kuvakaappaus | `S` |
| Soittolista | `F9` |
| Asetukset | `Ctrl+P` |

Jokaisen pikanäppäimen voi muuttaa kohdassa *Asetukset ▸ Näppäimistö ja hiiri*.
