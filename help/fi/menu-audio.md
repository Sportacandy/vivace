# Ääni

**Ääni**-valikko ohjaa ääntä.

- **Raita** — valitse tiedoston upotetuista ääniraidoista. Ensisijaisen kielen voi
  valita automaattisesti kohdassa *Asetukset ▸ Yleiset*.
- **Mykistä** (`M`) — kytke ääni päälle/pois.
- **Hiljemmalle** / **Kovemmalle** (`↓` / `↑`) — muuta äänenvoimakkuutta; askelen
  koko asetetaan kohdassa *Asetukset ▸ Yleiset*.
- **Viive −** / **Viive +** — siirrä ääntä aikaisemmaksi tai myöhemmäksi huulisynkronoinnin korjaamiseksi.
- **Aseta viive…** — syötä tarkka tiedostokohtainen äänen viive.

## Tietoja äänen viiveestä

Tiedostoon sovellettava viive on **laitekohtaisen** viiveen (muistetaan kullekin
äänen lähtölaitteelle — hyödyllinen Bluetooth-viiveessä, asetetaan kohdassa
*Asetukset ▸ Yleiset ▸ Audio*) ja tämän valikon **tiedostokohtaisen** viiveen
summa. Vivace soveltaa siirtymän viivyttämällä videota vastaamaan, joten
negatiiviset viiveet (ääni tulisi aikaisemmin) toimivat odotetusti.
