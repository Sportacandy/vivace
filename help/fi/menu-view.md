# Näytä

**Näytä**-valikko näyttää tietoja ja toissijaisia paneeleja.

- **Tiedot ja ominaisuudet…** — yksityiskohtainen valintaikkuna nykyisestä
  mediasta: yleistiedot, kanavoinnin purkaja sekä video-/äänikoodekit (Vivacen
  taustajärjestelmän tukemat muodot korostettuina). Vain tiedoksi.
- **Soittolista** (`F9`) — näytä tai piilota soittolista. Kohdan
  *Asetukset ▸ Käyttöliittymä* mukaan se näkyy kiinnitettynä paneelina tai
  erillisenä ikkunana. Katso [Avaa](menu-open.md) soittolistojen lataamiseen.
- **OSD** — videon päällä näytettävä näyttö:
  - yksityiskohtien taso (vain tekstitys; äänenvoimakkuus + kelaus;
    äänenvoimakkuus + kelaus + ajastin) laajenee; OSD:n kesto ja fonttikoko ovat
    kohdassa *Asetukset ▸ Yleiset*.
