# Ohje

**Ohje**-valikko.

- **Sisältö** (`F1`) — avaa tämä ohjeikkuna. Vasemmalla oleva luettelo on
  sisällysluettelo; napsauta aihetta lukeaksesi sen oikealla. Sivun sisäiset
  linkit avaavat liittyvän aiheen tähän; ulkoiset linkit avautuvat selaimessasi.
- **Tarkista päivitykset** — katso, onko uudempi Vivace saatavilla. Automaattiset
  tarkistukset voi ottaa käyttöön kohdassa *Asetukset ▸ Päivitykset*.
- **Asenna / päivitä YouTube-tuki** — lataa tai päivitä `yt-dlp`, apuohjelma, jota
  Vivace käyttää YouTube-linkkien toistoon (katso *Asetukset ▸ Verkko*).
- **Tietoja Vivacesta** — versio, Qt-versiot, kiitokset ja lisenssi.

Vivace on alusta asti rakennettu, täysin Qt-pohjainen mediasoitin, joka on saanut
inspiraationsa SMPlayerista, ja se julkaistaan **GPL-3.0-or-later**-lisenssillä.
