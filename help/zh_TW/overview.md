# 入門

歡迎使用 **Vivace**（讀作 *vee-VAH-cheh*），一款純 Qt 打造的快速媒體播放器。這是
Vivace 內建的離線說明。請從左側清單中選擇一個主題，或先從下面的基礎內容開始。

## 播放內容

- **檔案** — *開啟 ▸ 檔案…*（`Ctrl+O`），或將檔案拖到視窗上。
- **資料夾** — *開啟 ▸ 目錄…* 播放資料夾中的所有內容。
- **URL / 串流** — *開啟 ▸ URL…*（`Ctrl+U`）。
- **DVD** — *開啟 ▸ 光碟 ▸ DVD…*，或拖入包含 `VIDEO_TS` 的資料夾。

一次拖入多個檔案會將它們作為播放清單載入。

## 基本快速鍵

| 動作 | 快速鍵 |
| --- | --- |
| 播放 / 暫停 | `空格` |
| 倒轉 / 快轉 | `←` / `→` |
| 減小 / 增大音量 | `↓` / `↑` |
| 靜音 | `M` |
| 全螢幕 | `F`（或按兩下影片） |
| 離開全螢幕 | `Esc` |
| 擷取螢幕截圖 | `S` |
| 播放清單 | `F9` |
| 偏好設定 | `Ctrl+P` |

每個快速鍵都可以在 *偏好設定 ▸ 鍵盤與滑鼠* 中變更。
