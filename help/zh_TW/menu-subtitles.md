# 字幕

**字幕** 選單管理字幕。

- **音軌** — 選擇內嵌字幕軌，或選擇 **關閉** 以隱藏字幕。
- **載入字幕…** — 載入外部 `.srt`、`.vtt` 或 `.ass`/`.ssa` 檔案。Vivace 也可以自動
  載入影片旁邊的字幕（*偏好設定 ▸ 字幕*）。
- **在 OpenSubtitles 尋找字幕…** — 線上搜尋並下載字幕。這需要一個 OpenSubtitles
  API 金鑰，在 *偏好設定 ▸ 網路* 中輸入。
- **卸載字幕** — 移除目前載入的外部字幕。
- **延遲 −** / **延遲 +** — 提前或延後字幕的時間軸。
- **設定延遲…** — 輸入精確的字幕延遲。

字幕的字型、大小與位置在 *偏好設定 ▸ 字幕* 中設定。

**注意：** 外部載入的字幕透過 *載入* / *卸載* 控制 — 它不會列在 *音軌* 下（那裡顯示
內嵌音軌與 *關閉*）。
