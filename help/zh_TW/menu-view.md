# 檢視

**檢視** 選單顯示資訊與輔助面板。

- **資訊與屬性…** — 關於目前媒體的詳細對話方塊：一般資訊、解多工器以及視訊/音訊編解碼器
  （突顯 Vivace 後端支援的格式）。僅供參考。
- **播放清單**（`F9`）— 顯示或隱藏播放清單。視 *偏好設定 ▸ 介面* 而定，它會顯示為停駐
  面板或獨立視窗。有關載入播放清單，請參閱 [開啟](menu-open.md)。
- **OSD** — 顯示在影片上方的螢幕顯示：
  - 詳細程度（僅字幕；音量 + 定位；音量 + 定位 + 計時器）正在擴充；OSD 持續時間與字型
    大小在 *偏好設定 ▸ 一般* 中。
