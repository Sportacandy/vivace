# 視訊

**視訊** 選單調整畫面的顯示方式。這些設定套用於目前檔案，並為每個新檔案重設。

- **音軌** — 在檔案的內嵌視訊軌中選擇。
- **全螢幕**（`F`）— 切換全螢幕；`Esc` 離開。您也可以按兩下影片。
- **尺寸** — 將視窗調整為影片原始解析度的 **50% / 100% / 200%**。
- **縮放與平移**
  - **重設**（`Shift+E`）、**縮小**（`W`）、**放大**（`E`）。
  - **向左 / 右 / 上 / 下移動**（`Alt`+方向鍵）以平移放大的畫面。
- **長寬比** — 強制指定顯示長寬比（自動、4:3、16:9……）。
- **旋轉** — 無、順時針 90°、逆時針 90° 或 180°。
- **翻轉畫面** / **鏡像畫面** — 垂直或水平翻轉。
- **等化器…**（`Ctrl+E`）— 亮度、對比、色相、飽和度與 Gamma。
  *設為預設值* 會將您的設定套用於日後的檔案。
- **截圖**（`S`）— 儲存目前畫格。資料夾與影像格式在 *偏好設定 ▸ 一般* 中設定。
