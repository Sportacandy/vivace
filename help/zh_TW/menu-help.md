# 說明

**說明** 選單。

- **內容**（`F1`）— 開啟此說明視窗。左側清單是目錄；按一下某個主題即可在右側閱讀。頁面內
  的連結會在此處開啟相關主題；外部連結會在您的網頁瀏覽器中開啟。
- **檢查更新** — 查看是否有更新的 Vivace 版本。可在 *偏好設定 ▸ 更新* 中啟用自動檢查。
- **安裝 / 更新 YouTube 支援** — 下載或更新 `yt-dlp`，這是 Vivace 用來播放 YouTube
  連結的輔助程式（見 *偏好設定 ▸ 網路*）。
- **關於 Vivace** — 版本、Qt 版本、致謝與授權。

Vivace 是一款從零打造、純 Qt 的媒體播放器，受 SMPlayer 啟發，在
**GPL-3.0-or-later** 授權下發布。
