# Per iniziare

Benvenuti in **Vivace** (pronunciato *vi-VA-che*), un lettore multimediale
veloce e in puro Qt. Questa è la guida offline integrata di Vivace. Scegliete un
argomento dall'elenco a sinistra oppure iniziate dalle basi qui sotto.

## Riprodurre qualcosa

- **Un file** — *Apri ▸ File…* (`Ctrl+O`), oppure trascinate un file sulla finestra.
- **Una cartella** — *Apri ▸ Cartella…* riproduce tutto ciò che si trova in una cartella.
- **Un URL / flusso** — *Apri ▸ URL…* (`Ctrl+U`).
- **Un DVD** — *Apri ▸ Disco ▸ DVD…*, oppure trascinate una cartella contenente `VIDEO_TS`.

Trascinando più file insieme, vengono caricati come playlist.

## Scorciatoie essenziali

| Azione | Scorciatoia |
| --- | --- |
| Riproduci / pausa | `Spazio` |
| Indietro / avanti | `←` / `→` |
| Volume giù / su | `↓` / `↑` |
| Muto | `M` |
| Schermo intero | `F` (o doppio clic sul video) |
| Esci da schermo intero | `Esc` |
| Cattura schermata | `S` |
| Playlist | `F9` |
| Preferenze | `Ctrl+P` |

Ogni scorciatoia può essere modificata in *Preferenze ▸ Tastiera e mouse*.

Ogni menu di Vivace ha la propria pagina:

- [Video](menu-video.md) — schermo intero, dimensione, zoom, aspetto, equalizzatore, schermate
- [Audio](menu-audio.md) — tracce, volume, ritardo
- [Sottotitoli](menu-subtitles.md) — tracce, file esterni, sincronizzazione
- [Sfoglia](menu-browse.md) — titoli DVD, capitoli, segnalibri
- [Visualizza](menu-view.md) — info file, playlist, OSD
- [Opzioni](menu-options.md) — preferenze, icona nella barra, barre degli strumenti, barra di stato
- [Aiuto](menu-help.md) — questa guida, aggiornamenti, informazioni
