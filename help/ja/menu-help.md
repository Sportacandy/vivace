# ヘルプ

**ヘルプ** メニューには、Vivace に関する情報と補助機能があります。

- **目次**（`F1`）— このヘルプウィンドウを開きます。左に目次、右に本文が表示され、
  各メインメニューの説明を読めます。
- **YouTube サポートをインストール／更新** — 公式リリースから `yt-dlp` を
  ダウンロードして設定します（YouTube 再生に必要。既定では無効）。
- **更新を確認** — 新しいバージョンがあるか調べます。定期的な自動確認は
  *設定 ▸ 更新* で設定します。
- **Vivace について** — バージョン情報、使用している Qt のバージョン、
  クレジット、ライセンス（GPL-3.0-or-later）を表示します。

## Vivace について

Vivace は SMPlayer に着想を得た、純粋な Qt 製のメディアプレーヤーです。mplayer や
mpv のプロセスバックエンドを使わず、Qt Multimedia（FFmpeg バックエンド）で再生します。
アイコンテーマや UI の慣習の多くは SMPlayer から GPL のもとで受け継いでいます。
