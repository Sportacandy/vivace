# コマンドラインオプション

Vivace はターミナルから、メディアファイルといくつかのオプションを指定して起動でき
ます。GNU 形式のロングオプションを使います（短い別名はかっこ内に示します）。

```
vivace [オプション] [メディア]
```

- **メディア** — 再生するファイルパスまたは URL。Vivace が通常開けるもの（ローカル
  ファイル、フォルダー、`VIDEO_TS`、ネットワークストリーム、YouTube の URL）を
  指定できます。

## オプション

- **`--help`（`-h`）** — オプションの一覧を表示して終了します。
- **`--version`（`-v`）** — バージョンを表示して終了します。
- **`--fullscreen`（`-f`）** — 全画面表示で起動します。
- **`--ontop`（`-t`）** — ウィンドウを常に最前面に表示します。
- **`--close-at-end`（`-c`）** — 再生が終わったら Vivace を閉じます。
- **`--sub <ファイル>`（`-s`）** — 指定した字幕ファイルを読み込みます。
- **`--start <時間>`（`-b`）** — `<時間>` から再生を開始します。`h:m:s`、`m:s`、
  または秒数で指定します。
- **`--pos <x,y>`** — ウィンドウを画面上の位置 `x,y` に配置します。
- **`--size <w,h>`** — ウィンドウのサイズを `w`×`h` にします。

## 例

```
vivace movie.mkv
vivace --fullscreen --start 1:30 movie.mkv
vivace -f -b 90 -s movie.en.srt movie.mkv
vivace --pos 100,100 --size 1280,720 movie.mp4
```

**注（Windows）:** Vivace は GUI アプリケーションのため、`--help` / `--version` は
起動元のターミナルに出力されます（先にシェルのプロンプトが戻ることがあり、テキストが
その後に表示される場合があります）。シングルインスタンスモード（*設定 ▸ インター
フェース ▸ インスタンス*）が有効なときは、メディアファイルとこれらのオプションを
付けて再度起動すると、既に動作中のウィンドウに適用されます。
