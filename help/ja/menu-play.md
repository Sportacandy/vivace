# 再生

**再生** メニューは、再生とシークを操作します。

## 操作

- **再生**、**一時停止**、**停止** — `Space` で再生／一時停止を切り替え、
  `Ctrl+Space` で停止します。
- **コマ送り**（`.`）と **コマ戻し** — 1 フレームずつ進めたり戻したりします。
- **前へ** / **次へ** — プレイリストの項目を移動します。

## シーク

前後それぞれに 3 段階のシーク幅（**巻き戻し** / **早送り**）があり、現在の秒数が
表示されます。刻み幅は *設定 ▸ インターフェース ▸ シーク* で調整します。`←` / `→`、
シークスライダー、映像上のマウスホイールでもシークできます。

## 速度

- **標準速度**（`Backspace`）、**速度を半分に**、**速度を 2 倍に**。
- **速度 -10% / +10%**、さらに細かい ±4% / ±1% の刻み。
- **ピッチ補正** — 標準以外の速度でも声を自然に保ちます。

再生速度はファイルごとで、新しいファイルは標準速度で始まります。

## A-B 区間

ファイルの一部をループ再生します。

- **A マーカーを設定**、**B マーカーを設定** — 始点と終点を指定します。
- **リピート** — A-B 区間をループします（これがオンのときだけループします）。
- **A-B マーカーをクリア** — マーカーを削除します。マーカーはファイルごとにリセット
  されます。

**指定位置へ移動…**（特定の時間へジャンプ）は今後のリリースで予定しています。

## キャスト

**キャスト ▸ スマートフォン/タブレット…** は小さな内蔵 Web サーバーを起動し、
同じネットワーク上のスマートフォンやタブレットがページを開いて、Vivace が
現在再生しているファイルを再生できるようにします。QR コードをスキャンする
か、表示されたアドレスを入力してください。ポートは *設定 ▸ ネットワーク ▸
キャスト* で固定されており（キャストのたびに編集するものではありません）、
ファイアウォールやルーターで一度だけ許可すれば済みます。配信されるのは現在
再生中のファイルのみで、キャストがオンの間だけです。
