# ブラウズ

**ブラウズ** メニューは、メディア内を移動します。項目は、扱っているファイルや
ディスクに応じて変わります。

## チャプター

ファイル（Matroska/WebM や MP4/MOV に埋め込まれたもの）とディスクの両方のチャプター
を一覧します。クリックしてそのチャプターへジャンプします。コントロールバーの
前へ／次のチャプターボタンや、シークバーの目盛りでも移動できます。

## DVD

- **タイトル** — DVD のタイトルを一覧します（IFO から取得した長さ付き）。ふつうは
  最も長いタイトルが本編です。
- **DVD メニュー** — ディスクのメニューを表示します（**開く ▸ 設定** でメニュー機能を
  有効にしている場合）。矢印キーとマウスでボタンを操作します。

## ブックマーク

再生位置に名前を付けて保存します（ファイルごと、DVD はタイトルごと）。

- **ブックマークを追加**（`Ctrl+A`）— 現在の位置を保存します。
- **ブックマークを編集…** — 名前と時間の一覧を編集します。
- 保存したブックマークをクリックすると、その位置へシークします。

ブックマークは再開データとは別に保存されるため、再開履歴を消しても残ります。
