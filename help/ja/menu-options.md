# オプション

**オプション** メニューからは、設定やアプリの各種調整にアクセスします。

- **設定…**（`Ctrl+P`）— メインの設定ダイアログを開きます。左側の一覧から
  各セクション（一般、インターフェース、プレイリスト、字幕、ネットワーク、
  テレビとラジオ、キーボードとマウス、更新、ファイルの種類、詳細）へ移動します。
  変更は即座に反映され、**キャンセル** で元に戻ります。
  - **ネットワーク** には OpenSubtitles・YouTube・プロキシ・キャストの各タブが
    あります。**プロキシ** タブでは、任意の HTTP または SOCKS5 プロキシをアプリ
    ケーション全体（OpenSubtitles 検索・アップデート確認、および HTTP の場合の
    みメディア再生と yt-dlp）に適用できます。**キャスト** タブでは、*再生 ▸
    キャスト ▸ スマートフォン/タブレット* が待ち受けるポートを固定します。
    OpenSubtitles アカウントのパスワードとプロキシのパスワードは、Vivace 自体の
    設定ファイルにではなく、お使いのオペレーティングシステムの資格情報マネー
    ジャーに安全に保存されます。
- **システムトレイにアイコンを表示** — トレイアイコンを切り替えます。閉じたときに
  トレイへ最小化できます。
- **ツールバー** — メインツールバーとコントロールバーの表示切り替えと、
  レイアウト編集（**メインツールバーを編集** / **コントロールバーを編集**）。
- **ステータスバー** — ステータスバーの表示切り替えと、表示する情報（映像・音声情報、
  形式、ビットレート、フレームカウンター、時間表示）の選択。

## キーボードとマウス

*設定 ▸ キーボードとマウス* で、ほとんどの操作のショートカットを変更できます。
ショートカットをクリックして新しいキーの組み合わせを押します。競合は検出され、
一覧の絞り込みや、個別・一括のリセットもできます。編集は即座に反映・保存されます
（設定の OK／キャンセルの対象外です）。

同じページで、マウスの左クリック・ダブルクリック・中クリック・ホイールの動作も
割り当てられます。

## YouTubeダウンロード用のCookieのエクスポート

**Cookie ファイル:** フィールド（*設定 ▸ ネットワーク ▸ YouTube*）を使うと、
**ダウンロードして再生** および **外部ツール** の YouTube モードで、サインイン
した状態であるかのように動作させることができます。年齢制限付き・メンバー限定
などアカウント認証が必要な動画に必要で、フル HD/4K のダウンロードを可能にする
ものでもあります。このフィールドには、従来からある Netscape 形式の平文
`cookies.txt` ファイル（yt-dlp 自体の `--cookies` オプションが読み込むのと同じ
形式）を指定します。Vivace はブラウザのプロファイルから直接 Cookie を読み取る
ことはありません。

**作成手順:**

1. 普段使っているブラウザで youtube.com にサインインします。使用したいアクセ
   ス権を持つアカウントでログインしてください。
2. Netscape 形式で書き出せる Cookie エクスポート用のブラウザ拡張機能をインス
   トールします。Chrome、Edge、Brave では、ブラウザの拡張機能ストアで
   「Get cookies.txt」のようなものを検索してください。Firefox では
   「cookies.txt」を検索してください。従来の Netscape `cookies.txt` 形式で
   書き出すと明記されている拡張機能であれば、どれでも使用できます。
3. youtube.com のタブを開いた状態で、その拡張機能を使ってそのサイトの
   Cookie をエクスポートし、結果をディスク上のどこかに `.txt` ファイルとして
   保存します。
4. Vivace で *設定 ▸ ネットワーク ▸ YouTube* を開き、**Cookie ファイル:** の
   横にある **参照…** を使ってそのファイルを選択します。

**注意事項:**

- `cookies.txt` ファイルは、事実上保存されたログインセッションです。この
  ファイルを持っている人は誰でも、Cookie が失効するかサインアウトするまで、
  あなたの YouTube アカウントとして操作できてしまいます。人目につかない場所
  に保管し、他人と共有しないでください。
- Cookie が使われるのは **ダウンロード** 系の経路（ダウンロードして再生／
  外部ツール）のみです。Vivace は **ストリーミング** モードでは意図的に
  Cookie を送信しません — サインイン済みのストリーム URL はそのセッションに
  紐付けられており、Vivace の単純な動画プレーヤーでは開けない形式になって
  しまうためです。そのため、Cookie ファイルが設定されていても、ストリーミ
  ングは匿名のまま行われます。
- Cookie には有効期限があります。以前は機能していたダウンロードが失敗したり、
  画質が低い/非会員向けの結果にフォールバックし始めた場合は、新しい
  `cookies.txt` を書き出し直してください。
