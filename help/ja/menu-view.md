# 表示

**表示** メニューは、ウィンドウの外観と情報の表示を切り替えます。

- **プレイリスト**（`F9`）— プレイリストパネルの表示／非表示を切り替えます。
  ドッキング表示にするか別ウィンドウにするかは *設定 ▸ インターフェース* で選べます。
- **OSD** — オンスクリーン表示を切り替えます。位置・情報量を選べます。表示時間と
  フォントサイズは *設定 ▸ インターフェース* で設定します。
- **メディア情報**（`Ctrl+I`）— 開いているファイルの詳細（コンテナ、映像・音声
  コーデック、解像度、ビットレートなど）を表示します。
- **GUI** — レイアウトを切り替えます。
  - **ベーシック** — メニュー＋ツールバー＋コントロールバー＋ステータスバー。
  - **ミニ** — コンパクトなコントロール列のみ。
  - **MPC** — 全幅のシークバーと MPC 風のボタン列。

プレイリストパネルではファイルの追加・削除・並べ替え・検索や、`.m3u` / `.m3u8`
プレイリストの読み込み・保存ができます。リピートとシャッフルもここで切り替えます。
