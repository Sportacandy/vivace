# 字幕

**字幕** メニューは、字幕を管理します。

- **トラック** — 埋め込み字幕トラックを選ぶか、**オフ** で字幕を隠します。
- **字幕を読み込む…** — 外部の `.srt`・`.vtt`・`.ass`/`.ssa` ファイルを読み込みます。
  動画の隣にある字幕を自動読み込みすることもできます（*設定 ▸ 字幕*）。
- **OpenSubtitles で字幕を検索…** — オンラインで字幕を検索・ダウンロードします。
  OpenSubtitles の API キーが必要です（*設定 ▸ ネットワーク* で入力）。
- **字幕を閉じる** — 現在読み込んでいる外部字幕を解除します。
- **遅延 -** / **遅延 +** — 字幕のタイミングを早める／遅らせます。
- **遅延を設定…** — 正確な字幕遅延を入力します。

字幕のフォント・サイズ・位置は *設定 ▸ 字幕* で設定します。

**注:** 読み込んだ外部字幕は **読み込む** / **閉じる** で操作します。
**トラック**（埋め込みトラックと **オフ** を一覧）には表示されません。
