# はじめに

**Vivace**（ヴィヴァーチェ）へようこそ。軽快な、純 Qt 製メディアプレーヤーです。
これは Vivace に組み込まれた、オフラインで読めるヘルプです。左の一覧から
トピックを選ぶか、まずは下の基本から始めてください。

## 再生してみる

- **ファイル** — *開く ▸ ファイル…*（`Ctrl+O`）、またはウィンドウにファイルを
  ドラッグ＆ドロップします。
- **フォルダー** — *開く ▸ フォルダー…* でフォルダー内をすべて再生します。
- **URL / ストリーム** — *開く ▸ URL…*（`Ctrl+U`）。
- **DVD** — *開く ▸ ディスク ▸ DVD…*、または `VIDEO_TS` を含むフォルダーを
  ドロップします。

複数のファイルを一度にドロップすると、プレイリストとして読み込まれます。

## 主なショートカット

| 操作 | ショートカット |
| --- | --- |
| 再生 / 一時停止 | `Space` |
| 巻き戻し / 早送り | `←` / `→` |
| 音量 下げる / 上げる | `↓` / `↑` |
| ミュート | `M` |
| 全画面表示 | `F`（または映像をダブルクリック） |
| 全画面表示の解除 | `Esc` |
| スクリーンショット | `S` |
| プレイリスト | `F9` |
| 設定 | `Ctrl+P` |

すべてのショートカットは *設定 ▸ キーボードとマウス* で変更できます。
