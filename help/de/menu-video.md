# Video

Das Menü **Video** passt die Bilddarstellung an. Diese Einstellungen gelten für
die aktuelle Datei und werden für jede neue zurückgesetzt.

- **Spur** — unter den eingebetteten Videospuren der Datei wählen.
- **Vollbild** (`F`) — Vollbild umschalten; `Esc` verlässt es. Sie können auch
  auf das Video doppelklicken.
- **Größe** — das Fenster auf **50 % / 100 % / 200 %** der nativen Auflösung des
  Videos anpassen.
- **Zoom und Verschieben**
  - **Zurücksetzen** (`Umschalt+E`), **Zoom −** (`W`), **Zoom +** (`E`).
  - **Nach links / rechts / oben / unten verschieben** (`Alt`+Pfeiltasten), um ein
    gezoomtes Bild zu verschieben.
- **Seitenverhältnis** — ein Anzeigeverhältnis erzwingen (Auto, 4:3, 16:9, …).
- **Drehen** — Keine, 90° im Uhrzeigersinn, 90° gegen den Uhrzeigersinn oder 180°.
- **Bild spiegeln (vertikal)** / **Bild spiegeln (horizontal)** — vertikal oder
  horizontal spiegeln.
- **Equalizer…** (`Strg+E`) — Helligkeit, Kontrast, Farbton, Sättigung und Gamma.
  *Als Standardwerte festlegen* wendet Ihre Einstellungen auf künftige Dateien an.
- **Screenshot** (`S`) — das aktuelle Bild speichern. Ordner und Bildformat
  werden unter *Einstellungen ▸ Allgemein* festgelegt.
