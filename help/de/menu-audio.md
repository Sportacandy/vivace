# Ton

Das Menü **Ton** steuert den Klang.

- **Spur** — unter den eingebetteten Tonspuren der Datei wählen. Eine bevorzugte
  Sprache kann unter *Einstellungen ▸ Allgemein* automatisch gewählt werden.
- **Stummschalten** (`M`) — Ton ein-/ausschalten.
- **Leiser** / **Lauter** (`↓` / `↑`) — die Lautstärke ändern; die Schrittweite
  wird unter *Einstellungen ▸ Allgemein* festgelegt.
- **Verzögerung −** / **Verzögerung +** — den Ton früher oder später verschieben,
  um die Lippensynchronität zu korrigieren.
- **Verzögerung festlegen…** — eine genaue Tonverzögerung je Datei eingeben.

## Über die Tonverzögerung

Die auf eine Datei angewandte Verzögerung ist die Summe aus einer
**geräteabhängigen** Verzögerung (für jedes Tonausgabegerät gemerkt — nützlich
bei Bluetooth-Latenz, unter *Einstellungen ▸ Allgemein ▸ Audio* festgelegt) und
der **dateiabhängigen** Verzögerung aus diesem Menü. Vivace wendet den Versatz
an, indem es das Video entsprechend verzögert, sodass negative Verzögerungen
(Ton soll früher kommen) wie erwartet funktionieren.
