# Wiedergabe

Das Menü **Wiedergabe** steuert die Wiedergabe und das Suchen.

## Steuerung

- **Wiedergabe**, **Pause**, **Stopp** — `Leertaste` schaltet zwischen
  Wiedergabe/Pause um; `Strg+Leertaste` stoppt.
- **Einzelbild vor** (`.`) und **Einzelbild zurück** — ein Bild vor oder zurück.
- **Vorheriges** / **Nächstes** — zwischen Wiedergabelisten-Einträgen wechseln.

## Suchen

In beide Richtungen werden drei Sprungweiten angeboten (**Zurück** / **Vor**),
mit ihrer aktuellen Dauer angezeigt. Passen Sie die Sprungweiten unter
*Einstellungen ▸ Oberfläche ▸ Suchen* an. Sie können auch mit `←` / `→`, dem
Suchregler oder dem Mausrad über dem Video suchen.

## Geschwindigkeit

- **Normale Geschwindigkeit** (`Rücktaste`), **Geschwindigkeit halbieren**,
  **Geschwindigkeit verdoppeln**.
- **Geschwindigkeit −10 % / +10 %** und feinere Schritte von ±4 % / ±1 %.
- **Tonhöhenkorrektur** — hält Stimmen bei abweichender Geschwindigkeit natürlich.

Die Wiedergabegeschwindigkeit gilt je Datei und wird für jede neue Datei zurückgesetzt.

## A-B-Abschnitt

Einen Teil einer Datei wiederholen:

- **A-Marke setzen**, **B-Marke setzen** — Anfang und Ende markieren.
- **Wiederholen** — den A-B-Bereich wiederholen (der Abschnitt wird nur
  wiederholt, solange dies aktiv ist).
- **A-B-Marken löschen** — sie entfernen. Marken werden für jede neue Datei zurückgesetzt.

**Springe zu…** (zu einer bestimmten Zeit) ist für eine spätere Version geplant.

## Übertragung

**Übertragen zum ▸ Smartphone/Tablet…** startet einen kleinen eingebauten
Webserver, damit ein Smartphone oder Tablet im selben Netzwerk eine Seite
öffnen und die Datei abspielen kann, die Vivace gerade wiedergibt — per
QR-Code oder durch Eingabe der angezeigten Adresse. Der Port ist unter
*Einstellungen ▸ Netzwerk ▸ Übertragung* fest eingestellt (nicht bei jeder
Sitzung änderbar), damit Sie ihn einmalig in Ihrer Firewall/Ihrem Router
freigeben können. Es wird nur die gerade abgespielte Datei übertragen, und
nur solange die Übertragung aktiv ist.
