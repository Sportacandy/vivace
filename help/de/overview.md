# Erste Schritte

Willkommen bei **Vivace** (ausgesprochen *wi-WA-tsche*), einem schnellen,
reinen Qt-Mediaplayer. Dies ist die integrierte Offline-Hilfe von Vivace. Wählen
Sie links ein Thema aus oder beginnen Sie mit den Grundlagen unten.

## Etwas abspielen

- **Eine Datei** — *Öffnen ▸ Datei…* (`Strg+O`) oder ziehen Sie eine Datei auf das Fenster.
- **Einen Ordner** — *Öffnen ▸ Verzeichnis…* spielt alles in einem Ordner ab.
- **Eine URL / einen Stream** — *Öffnen ▸ URL…* (`Strg+U`).
- **Eine DVD** — *Öffnen ▸ Disc ▸ DVD…* oder legen Sie einen Ordner mit `VIDEO_TS` ab.

Wenn Sie mehrere Dateien auf einmal ablegen, werden sie als Wiedergabeliste geladen.

## Wichtige Tastenkürzel

| Aktion | Tastenkürzel |
| --- | --- |
| Wiedergabe / Pause | `Leertaste` |
| Zurück / vor spulen | `←` / `→` |
| Leiser / lauter | `↓` / `↑` |
| Stummschalten | `M` |
| Vollbild | `F` (oder Doppelklick auf das Video) |
| Vollbild verlassen | `Esc` |
| Screenshot erstellen | `S` |
| Wiedergabeliste | `F9` |
| Einstellungen | `Strg+P` |

Jedes Tastenkürzel kann unter *Einstellungen ▸ Tastatur und Maus* geändert werden.
