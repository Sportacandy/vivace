# Kom godt i gang

Velkommen til **Vivace** (udtales *vi-VA-che*), en hurtig medieafspiller helt i
Qt. Dette er Vivaces indbyggede offlinehjælp. Vælg et emne i listen til venstre,
eller start med det grundlæggende nedenfor.

## Afspil noget

- **En fil** — *Åbn ▸ Fil…* (`Ctrl+O`), eller træk en fil hen på vinduet.
- **En mappe** — *Åbn ▸ Mappe…* afspiller alt i en mappe.
- **En URL / stream** — *Åbn ▸ URL…* (`Ctrl+U`).
- **En dvd** — *Åbn ▸ Disk ▸ Dvd…*, eller slip en mappe med `VIDEO_TS`.

At slippe flere filer på én gang indlæser dem som en afspilningsliste.

## Vigtige genveje

| Handling | Genvej |
| --- | --- |
| Afspil / pause | `Mellemrum` |
| Tilbage / frem | `←` / `→` |
| Skru ned / op | `↓` / `↑` |
| Slå lyd fra | `M` |
| Fuldskærm | `F` (eller dobbeltklik på videoen) |
| Forlad fuldskærm | `Esc` |
| Tag et skærmbillede | `S` |
| Afspilningsliste | `F9` |
| Indstillinger | `Ctrl+P` |

Alle genveje kan ændres i *Indstillinger ▸ Tastatur og mus*.

## Hjælp efter menu

Hver menu i Vivace har sin egen side:

- [Åbn](menu-open.md) — åbn filer, mapper, diske, URL'er, favoritter
- [Afspil](menu-play.md) — afspilning, søgning, hastighed, A-B-gentagelse
- [Video](menu-video.md) — fuldskærm, størrelse, zoom, billedforhold, equalizer, skærmbilleder
- [Lyd](menu-audio.md) — spor, lydstyrke, forsinkelse
- [Undertekster](menu-subtitles.md) — spor, eksterne filer, timing
- [Gennemse](menu-browse.md) — dvd-titler, kapitler, bogmærker
- [Vis](menu-view.md) — medieinfo, afspilningsliste, OSD
- [Indstillinger](menu-options.md) — indstillinger, bakkeikon, værktøjslinjer, statuslinje
- [Hjælp](menu-help.md) — denne hjælp, opdateringer, om
