# Aan de slag

Welkom bij **Vivace** (uitgesproken als *vie-VA-tsje*), een snelle mediaspeler
volledig in Qt. Dit is de ingebouwde offline-hulp van Vivace. Kies een onderwerp
in de lijst links, of begin hieronder met de basis.

## Iets afspelen

- **Een bestand** — *Openen ▸ Bestand…* (`Ctrl+O`), of sleep een bestand op het venster.
- **Een map** — *Openen ▸ Map…* speelt alles in een map af.
- **Een URL / stream** — *Openen ▸ URL…* (`Ctrl+U`).
- **Een dvd** — *Openen ▸ Schijf ▸ Dvd…*, of sleep een map met `VIDEO_TS`.

Meerdere bestanden tegelijk neerzetten laadt ze als een afspeellijst.

## Essentiële sneltoetsen

| Actie | Sneltoets |
| --- | --- |
| Afspelen / pauzeren | `Spatie` |
| Terug / vooruit spoelen | `←` / `→` |
| Volume omlaag / omhoog | `↓` / `↑` |
| Dempen | `M` |
| Volledig scherm | `F` (of dubbelklik op de video) |
| Volledig scherm verlaten | `Esc` |
| Schermafbeelding maken | `S` |
| Afspeellijst | `F9` |
| Voorkeuren | `Ctrl+P` |

Elke sneltoets kan worden gewijzigd in *Voorkeuren ▸ Toetsenbord en muis*.

## Hulp per menu

Elk menu van Vivace heeft zijn eigen pagina:

- [Openen](menu-open.md) — bestanden, mappen, schijven, URL's, favorieten openen
- [Afspelen](menu-play.md) — afspelen, zoeken, snelheid, A-B-herhaling
- [Video](menu-video.md) — volledig scherm, grootte, zoom, beeldverhouding, equalizer, schermafbeeldingen
- [Audio](menu-audio.md) — sporen, volume, vertraging
- [Ondertitels](menu-subtitles.md) — sporen, externe bestanden, timing
- [Bladeren](menu-browse.md) — dvd-titels, hoofdstukken, bladwijzers
- [Weergave](menu-view.md) — media-info, afspeellijst, OSD
- [Opties](menu-options.md) — voorkeuren, systeemvakpictogram, werkbalken, statusbalk
- [Help](menu-help.md) — deze hulp, updates, info
