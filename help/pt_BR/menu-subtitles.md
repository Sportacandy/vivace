# Legendas

O menu **Legendas** gerencia as legendas.

- **Faixa** — escolher uma faixa de legenda incorporada, ou **Desligado** para ocultá-las.
- **Carregar legendas…** — carregar um arquivo externo `.srt`, `.vtt` ou
  `.ass`/`.ssa`. O Vivace também pode carregar automaticamente uma legenda que
  esteja ao lado do vídeo (*Preferências ▸ Legendas*).
- **Buscar legendas no OpenSubtitles…** — buscar e baixar legendas on-line.
  Requer uma chave de API do OpenSubtitles, inserida em *Preferências ▸ Rede*.
- **Descarregar legendas** — remover a legenda externa carregada no momento.
- **Atraso −** / **Atraso +** — adiantar ou atrasar a sincronização das legendas.
- **Definir atraso…** — inserir um atraso de legenda exato.

A fonte, o tamanho e a posição das legendas se definem em *Preferências ▸ Legendas*.

**Nota:** uma legenda carregada externamente é controlada com *Carregar* /
*Descarregar* — ela não aparece em *Faixa* (que mostra as faixas incorporadas e *Desligado*).
