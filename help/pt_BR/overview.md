# Primeiros passos

Bem-vindo ao **Vivace** (pronunciado *vi-VÁ-tche*), um reprodutor de mídia
rápido e em Qt puro. Esta é a ajuda offline integrada do Vivace. Escolha um tópico
na lista à esquerda ou comece pelo básico abaixo.

## Reproduzir algo

- **Um arquivo** — *Abrir ▸ Arquivo…* (`Ctrl+O`), ou arraste um arquivo para a janela.
- **Uma pasta** — *Abrir ▸ Diretório…* reproduz tudo em uma pasta.
- **Uma URL / transmissão** — *Abrir ▸ URL…* (`Ctrl+U`).
- **Um DVD** — *Abrir ▸ Disco ▸ DVD…*, ou solte uma pasta contendo `VIDEO_TS`.

Soltar vários arquivos de uma vez os carrega como uma lista de reprodução.

## Atalhos essenciais

| Ação | Atalho |
| --- | --- |
| Reproduzir / pausar | `Espaço` |
| Retroceder / avançar | `←` / `→` |
| Diminuir / aumentar volume | `↓` / `↑` |
| Silenciar | `M` |
| Tela cheia | `F` (ou duplo clique no vídeo) |
| Sair da tela cheia | `Esc` |
| Fazer uma captura de tela | `S` |
| Lista de reprodução | `F9` |
| Preferências | `Ctrl+P` |

Todos os atalhos podem ser alterados em *Preferências ▸ Teclado e mouse*.
