# Vídeo

O menu **Vídeo** ajusta como a imagem é exibida. Estas configurações se aplicam
ao arquivo atual e são reiniciadas a cada novo.

- **Faixa** — escolher entre as faixas de vídeo incorporadas do arquivo.
- **Tela cheia** (`F`) — alternar tela cheia; `Esc` sai dela. Você também pode
  dar um duplo clique no vídeo.
- **Tamanho** — redimensionar a janela para **50% / 100% / 200%** da resolução
  nativa do vídeo.
- **Zoom e deslocamento**
  - **Redefinir** (`Shift+E`), **Zoom −** (`W`), **Zoom +** (`E`).
  - **Mover para esquerda / direita / cima / baixo** (`Alt`+setas) para deslocar uma imagem ampliada.
- **Proporção** — forçar uma proporção de exibição (Automático, 4:3, 16:9, …).
- **Girar** — Nenhuma, 90° horário, 90° anti-horário ou 180°.
- **Inverter imagem** / **Espelhar imagem** — inverter na vertical ou horizontal.
- **Equalizador…** (`Ctrl+E`) — brilho, contraste, matiz, saturação e gama.
  *Definir como valores padrão* aplica suas configurações a arquivos futuros.
- **Captura de tela** (`S`) — salvar o quadro atual. A pasta e o formato de
  imagem se definem em *Preferências ▸ Geral*.
