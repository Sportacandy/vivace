# Začínáme

Vítejte ve **Vivace** (vyslovuje se *ví-VA-če*), rychlém přehrávači médií čistě v
Qt. Toto je vestavěná offline nápověda Vivace. Vyberte téma ze seznamu vlevo nebo
začněte základy níže.

## Něco přehrát

- **Soubor** — *Otevřít ▸ Soubor…* (`Ctrl+O`), nebo přetáhněte soubor na okno.
- **Složku** — *Otevřít ▸ Adresář…* přehraje vše ve složce.
- **URL / stream** — *Otevřít ▸ URL…* (`Ctrl+U`).
- **DVD** — *Otevřít ▸ Disk ▸ DVD…*, nebo přetáhněte složku obsahující `VIDEO_TS`.

Přetažení více souborů najednou je načte jako seznam skladeb.

## Základní klávesové zkratky

| Akce | Zkratka |
| --- | --- |
| Přehrát / pozastavit | `Mezerník` |
| Zpět / vpřed | `←` / `→` |
| Ztlumit / zesílit | `↓` / `↑` |
| Ztlumit zvuk | `M` |
| Celá obrazovka | `F` (nebo dvojklik na video) |
| Opustit celou obrazovku | `Esc` |
| Pořídit snímek obrazovky | `S` |
| Seznam skladeb | `F9` |
| Předvolby | `Ctrl+P` |

Každou zkratku lze změnit v *Předvolby ▸ Klávesnice a myš*.

## Nápověda podle nabídky

Každá nabídka Vivace má vlastní stránku:

- [Otevřít](menu-open.md) — otevírání souborů, složek, disků, URL, oblíbených
- [Přehrávání](menu-play.md) — přehrávání, převíjení, rychlost, opakování A-B
- [Video](menu-video.md) — celá obrazovka, velikost, přiblížení, poměr stran, ekvalizér, snímky
- [Zvuk](menu-audio.md) — stopy, hlasitost, zpoždění
- [Titulky](menu-subtitles.md) — stopy, externí soubory, časování
- [Procházet](menu-browse.md) — tituly DVD, kapitoly, záložky
- [Zobrazit](menu-view.md) — informace o médiu, seznam skladeb, OSD
- [Možnosti](menu-options.md) — předvolby, ikona v oznamovací oblasti, panely nástrojů, stavový řádek
- [Nápověda](menu-help.md) — tato nápověda, aktualizace, o programu
