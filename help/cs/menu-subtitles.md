# Titulky

Nabídka **Titulky** spravuje titulky.

- **Stopa** — vybrat vloženou stopu titulků nebo **Vypnuto** pro jejich skrytí.
- **Načíst titulky…** — načíst externí soubor `.srt`, `.vtt` nebo `.ass`/`.ssa`.
  Vivace může také automaticky načíst titulky ležící vedle videa
  (*Předvolby ▸ Titulky*).
- **Najít titulky na OpenSubtitles…** — hledat a stahovat titulky online. Vyžaduje
  klíč API OpenSubtitles zadaný v *Předvolby ▸ Síť*.
- **Uvolnit titulky** — odstranit aktuálně načtené externí titulky.
- **Zpoždění −** / **Zpoždění +** — posunout časování titulků dříve nebo později.
- **Nastavit zpoždění…** — zadat přesné zpoždění titulků.

Písmo, velikost a poloha titulků se nastavují v *Předvolby ▸ Titulky*.

**Poznámka:** externě načtené titulky se ovládají přes *Načíst* / *Uvolnit* —
nejsou uvedeny v *Stopa* (kde jsou vložené stopy a *Vypnuto*).
