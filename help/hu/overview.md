# Első lépések

Üdvözli a **Vivace** (kiejtése *vi-VÁ-cse*), egy gyors, tisztán Qt-alapú
médialejátszó. Ez a Vivace beépített offline súgója. Válasszon témát a bal
oldali listából, vagy kezdje az alábbi alapokkal.

## Lejátszás

- **Fájl** — *Megnyitás ▸ Fájl…* (`Ctrl+O`), vagy húzzon egy fájlt az ablakra.
- **Mappa** — *Megnyitás ▸ Könyvtár…* lejátszik mindent egy mappában.
- **URL / adatfolyam** — *Megnyitás ▸ URL…* (`Ctrl+U`).
- **DVD** — *Megnyitás ▸ Lemez ▸ DVD…*, vagy ejtsen ide egy `VIDEO_TS`-t tartalmazó mappát.

Több fájl egyszerre ejtése lejátszási listaként tölti be őket.

## Alapvető billentyűparancsok

| Művelet | Billentyűparancs |
| --- | --- |
| Lejátszás / szünet | `Szóköz` |
| Vissza / előre | `←` / `→` |
| Halkítás / hangosítás | `↓` / `↑` |
| Némítás | `M` |
| Teljes képernyő | `F` (vagy kattintson duplán a videóra) |
| Kilépés a teljes képernyőből | `Esc` |
| Képernyőkép készítése | `S` |
| Lejátszási lista | `F9` |
| Beállítások | `Ctrl+P` |

Minden billentyűparancs módosítható a *Beállítások ▸ Billentyűzet és egér* menüben.

## Súgó menü szerint

A Vivace minden menüjének saját oldala van:

- [Megnyitás](menu-open.md) — fájlok, mappák, lemezek, URL-ek, kedvencek megnyitása
- [Lejátszás](menu-play.md) — lejátszás, tekerés, sebesség, A-B ismétlés
- [Videó](menu-video.md) — teljes képernyő, méret, nagyítás, képarány, hangszínszabályzó, képernyőképek
- [Hang](menu-audio.md) — sávok, hangerő, késleltetés
- [Felirat](menu-subtitles.md) — sávok, külső fájlok, időzítés
- [Tallózás](menu-browse.md) — DVD-címek, fejezetek, könyvjelzők
- [Nézet](menu-view.md) — médiainformációk, lejátszási lista, OSD
- [Beállítások menü](menu-options.md) — beállítások, tálcaikon, eszköztárak, állapotsor
- [Súgó](menu-help.md) — ez a súgó, frissítések, névjegy
